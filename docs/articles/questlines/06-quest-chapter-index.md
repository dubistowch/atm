# 任務章節一覽（FTB Quests）：本包任務書有哪些模組章節？

這份清單是從「任務書檔案」的繁中語系（`config/ftbquests/quests/lang/zh_tw`）整理出來的，回答最常見的問題：**FTB Quests 裡到底有推哪些模組/章節？**

> 提醒：章節名稱可能包含整合包作者的自訂章節（不一定等於模組名），而且同一模組可能分成多個章節。

---

## 怎麼用這張表（玩家版）
- 想找某模組：用瀏覽器/手機在本頁搜尋關鍵字（例如「沉浸工程」「新生魔藝」「自然靈氣」）。
- 想知道任務在哪：打開任務書後，用章節名稱或物品名搜尋。

## 下一步：要不要把「每章節有哪些任務」也整理成站內攻略？
如果你想把任務內容擴充成攻略文章，我可以再從：
- `config/ftbquests/quests/chapters/*.snbt`（章節結構/任務樹）
- `config/ftbquests/quests/lang/zh_tw/chapters/*.snbt`（繁中標題/描述，含圖示標記）
把每章節的關鍵任務節點抽出來，整理成「玩家視角下一步要做什麼」。
