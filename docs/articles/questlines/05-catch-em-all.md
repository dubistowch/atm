# 抓怪／圖鑑路線：化石、釣魚與追蹤（Catch ’Em All）

當主線在推產線與材料時，你很可能會想換個口味：去探險、抓稀有、補圖鑑。

`catch_em_all` 這章把「找怪」變成可規劃的流程，尤其聚焦：**考古化石**、**復活史前寶可夢**、**追蹤器**、**釣魚**。

對應 FTB Quests 章節：`catch_em_all`。

---

## 大目標
1. **完成一次完整循環**：找到史前地點 → 考古出化石 → 回基地復活
2. **把找怪從運氣變策略**：用追蹤器＋生態域/結構工具縮短瞎逛時間
3. **把探索成果收斂回基地**：戰利品能分類入庫，圖鑑能持續推進

---

## 進度清單（本章 10–25 項）
- [ ] 準備探險套裝：食物、床、火把、方塊、回家手段（/home 或傳送類方塊）
- [ ] 做一個「探險箱」或隨身收納，避免戰利品爆倉
- [ ] 準備考古工具（刷子等，依任務/JEI/REI）
- [ ] 找到第一個史前相關地點/結構並插路標
- [ ] 把史前地點做成「可重複跑的路線」（不要每次從零開始找）
- [ ] 蒐集到至少 1–2 種化石（先求有，重複很正常）
- [ ] 回基地搭建化石復活所需的多方塊結構（照任務圖/提示）
- [ ] 完成第一次化石復活（拿到第一隻史前寶可夢）
- [ ] 擴充復活區空間（後續你會想一次處理一批化石）
- [ ] 做出/取得寶可追蹤器並熟悉顯示方式（範圍/目標/地圖提示）
- [ ] 用追蹤器驗證生態域：這裡到底刷不刷你要的寶可夢
- [ ] 建立一份「缺口清單」：屬性/用途/圖鑑缺哪些，就去哪裡補
- [ ] 開始釣魚線：釣竿、補給、固定釣點（一次釣一輪再回家整理）
- [ ] 若有釣竿升級模板/強化流程，先做最划算的 1–2 個升級
- [ ] 把探索到的材料與寶可夢分類入庫（避免後面找不到）

## 你現在應該做什麼（依階段）

### 階段 A｜你想「先看到成果」（第一次跑通）
- 先找得到一個史前點位並插路標
- 先挖到第一批化石
- 回家先把復活結構做出來並成功復活一次

### 階段 B｜你想「穩定補圖鑑」（可重複）
- 把史前點位變成一條固定路線（例如 2–3 個點輪流跑）
- 追蹤器拿來做「地點驗證」：正確才投入時間

### 階段 C｜你想「把抓怪融入主線」（不卡倉庫）
- 探險成果回家就入庫（抽屜/分類箱/數位收納）
- 把稀有素材或掉落物納入你的產線/任務材料清單

---

## 資源採集／量產方法（通用，配方以 JEI/REI）

### 化石與考古材料
- **策略**：把史前地點當成「資源點」，不是一次性景點
- **效率**：
  - 找到一次就插路標
  - 回家清包再出門（避免爆倉）
  - 一次挖一批、一次復活一批（批次處理）

### 追蹤與找怪
- **追蹤器**：用來縮短「這裡到底刷不刷」的驗證成本
- **搭配工具**：Nature’s/Explorer’s Compass 類（找生態域/結構），再用追蹤器確認附近刷怪

### 釣魚
- 把釣魚當作「固定行程」：補給→釣一輪→回家分類
- 優先升級最影響效率的部件（咬鉤速度/耐久/掉落等，依整包設定）

---

## 常見卡關
- **找不到史前地點**：先用指南針類工具鎖定結構/生態域，再用地形判讀；找到一次就插路標
- **化石太少/一直重複**：把點位做成可重複路線；並用批次處理減少來回時間
- **復活結構不會擺**：先看任務圖或在測試世界試擺，確定方向與高度
- **釣魚很慢**：先把補給/收納流程順好，再追釣竿升級
