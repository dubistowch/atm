# 開荒與前期：Cobblemon 起手式（Getting Started）

前期的核心不是「挖到鑽石」，而是建立一個**可以每天重複、越跑越順**的循環：

> 出門抓怪／探索 → 對戰拿資源 → 回家治療、整理、補給 → 再出門

本篇對應 FTB Quests 章節：`getting_started`。

---

## 大目標
1. **把 Cobblemon 的三件事做成習慣**：圖鑑（找目標）／治療（續航）／PC（收納）
2. **讓抓怪變成可持續**：永遠有球、永遠有空位、永遠回得了家
3. **把消耗品「供應」做起來**：球果（杏果）與常用材料不再靠臨時去撿

---

## 進度清單（本章 10–25 項）
- [ ] 先完成生存基本盤：床、火把、食物、石/鐵工具
- [ ] 選定初始寶可夢並把牠練到能穩定打贏家附近野怪
- [ ] 確認鍵位：丟出/收回、隊伍介面、圖鑑（Pokédex）
- [ ] 做/取得圖鑑並學會查「生成環境/目標清單」
- [ ] 抓到 2–3 隻屬性互補的隊友（別追完美個體，先求覆蓋面）
- [ ] 找到一座寶可夢中心並設路標（當前期補給站）
- [ ] 在基地做出第一個固定治療來源（治療機或等效）
- [ ] 做出 PC（電腦）並建立盒子分類（主力/培育/收藏/待處理）
- [ ] 養成「隊伍永遠留 1–2 個空位」的抓怪習慣
- [ ] 做出至少一種隨身收納（背囊等）或建立「探險箱」回家再整理
- [ ] 開始蒐集球果/杏果（看到就拔、成熟就收），先存一箱
- [ ] 建一小塊球果種植區（先求有、再求美觀）
- [ ] 準備少量精靈球材料並維持「常備庫存」
- [ ] 做出牧場方塊（或任務要求的基地放養方塊）讓寶可夢在基地巡遊/協防
- [ ] 把基地周圍做成夜晚也安全：照明、圍牆或護城河（降低猝死成本）
- [ ] 建立「外出補給清單」：球、食物、方塊、火把、回家手段

## 你現在應該做什麼（依階段）

### 階段 A｜剛選完初始（0–1 小時）
- 先把生存做穩（床、火把、食物）
- 抓 2–3 隻屬性互補的野怪，先把隊伍湊齊
- 用圖鑑開始列「你想抓什麼」的短清單（工具人/坐騎/屬性缺口）

### 階段 B｜你開始常出門抓怪（1–4 小時）
- 把「治療」固定下來：先靠寶可夢中心，之後目標是自製治療機
- 做出 PC，解決「隊伍滿了又捨不得放生」的痛點
- 把球果/杏果當成你每天路過就要採的資源（像木頭一樣重要）

### 階段 C｜你要讓抓怪變得輕鬆（4 小時後）
- 牧場方塊/基地防衛到位，減少夜間干擾
- 開始思考「球的供應鏈」：杏果來源＋金屬來源＋加工流程（先半自動也行）

## 資源採集／量產方法（通用，配方以 JEI/REI）

### 球果／杏果（精靈球與部分消耗品核心）
- **採集**：看到球果樹就先插路標；成熟後右鍵收（依版本/模組設定）
- **種植**：先做小規模種植區，後續再擴成田
- **量產方向**（不綁模組）：
  - 用任意自動農場方案收成（收割→輸送→集中存放）
  - 或用你選的科技/魔法方案把作物變成自動產出

### 金屬（球殼/零件常用）
- 早期：礦挖＋熔爐
- 中期：礦物處理倍化（任一科技線）或改走資源作物/蜜蜂

### 治療與續航
- 前期：寶可夢中心（路標化）
- 中期：治療機（放在基地收納旁邊，讓「回家→治療→整理」變成一條線）

### 收納（讓你真的敢抓怪）
- 最低限度：分類箱＋告示牌/命名
- 建議：抽屜或數位收納（依你中期規劃）

## 常見卡關
- **一直輸對戰**：先補隊伍屬性覆蓋；同時把自己的人類裝備升到鐵裝/盾牌
- **隊伍爆滿、抓到好怪捨不得放**：PC 優先級很高；並養成分類習慣
- **找不到寶可夢中心**：看到白色混凝土大建築就插路標；找不到就先以自製治療機為目標
- **球果太少**：先以探索採集為主；中期有移動能力再拉大種植規模
