# 主線任務總覽：從開荒到「ATM 之星」

這系列文章用**玩家視角**帶你跑完整合包主線節奏：你不是在「照表抄作業」，而是在用任務當導航，把整包最重要的玩法（生存、探索、科技/魔法、自動化、終局目標）串成一條可走得完的路。

> 任務在本包中是「可選的」；不做也不會鎖內容。但主線非常適合當作：**下一步做什麼**、**缺什麼資源**、**要不要開始自動化** 的提示板。

---

## 大目標
- **短期（開荒～前期）**：有一個安全基地、基礎收納、能穩定補給與治療（Cobblemon）
- **中期（下界/終界）**：把關鍵維度打通、拿到中期材料與移動能力，開始選定你的主力模組路線
- **後期（量產/奇點/多模組整合）**：把資源從「手挖」變成「穩定產出」
- **終局（ATM 之星）**：把各模組產線整合成一張材料網，完成 **ATM Star**（把它當作整合包的「畢業證書」）

---

## 進度總清單（主線里程碑）

> 這份清單刻意寫成「你要做到什麼狀態」，而不是綁死某一種模組解法；配方與細節請以 **JEI/REI** 與任務顯示為準。

### Phase 1｜開荒（安全基地與基本循環）
- [ ] 設定重生點（床）並建立簡易防怪措施（火把/圍牆/門禁）
- [ ] 先完成「能活下來」的裝備：鐵裝（或等效）＋盾牌＋足夠食物
- [ ] 做出最基本收納：至少 4–8 個分類箱（礦物/木材/怪物掉落/任務雜物）
- [ ] 調好鍵位：地圖/路標、FTB Ultimine（連鎖挖礦）、Cobblemon 相關操作
- [ ] 建立回家與外出動線：路標/回程方式（如 /home 或傳送類方塊，依伺服器）

### Phase 2｜第一套自動化（從手作到穩定產出）
- [ ] 選定「第一套收納升級」：抽屜/數位收納（AE2 等）/其他物流（先做其中一種）
- [ ] 建立第一條可持續產出線：木材、食物、鐵（至少其中 1–2 種可半自動/自動）
- [ ] 供電/動力雛形（若走科技）：能讓機器連續運轉，不用一直手點
- [ ] 把常用材料變成「可補貨」：石頭、煤、紅石、基本金屬（用你選的方案）

### Phase 3｜下界／終界（中期門票）
- [ ] 安全化下界傳送門（封門、路標、回程通道）
- [ ] 取得下界關鍵資源（如烈焰粉、地獄石系材料、要塞資源等）
- [ ] 以「活著回來」為目標完成首次終界行程，並能穩定往返
- [ ] 中期移動能力到位（更快的移動/飛行/跨維度傳送，依你路線）

### Phase 4｜資源農場（量產底盤）
- [ ] 至少完成一套「量產底盤」：
  - [ ] Mystical Agriculture（種子→精華→資源）或
  - [ ] Resource Bees/樹/作物自動化 或
  - [ ] 礦物處理倍化工廠 或
  - [ ] 敵對掉落物自動化（刷怪/Hostile Networks 等）
- [ ] 讓產線材料能自動送回收納（或至少集中進同一區域）
- [ ] 開始處理「需要大量」的項目：壓縮、批量加工、批量儲存

### Phase 5｜ATM Star（終局整合）
- [ ] 把 ATM Star 材料表拆成「可量產」「要探索」「要多方塊結構/儀式」三類
- [ ] 優先完成最卡的 3–5 條材料線（通常是奇點/高階合金/儀式材料）
- [ ] 建立一個「星星專用看板」：缺什麼就回頭補哪條產線
- [ ] 完成 ATM Star（或達到只差少數材料、可穩定補齊的狀態）

## 你現在應該做什麼（依階段）

### 你剛進世界（0–2 小時）
- 先確保生存：食物、床、火把、基本工具
- 同步開始 Cobblemon 循環：抓怪→對戰→回家治療整理
- 不要急著跑太遠：先把「回得了家」當第一優先

### 你已經有鐵裝/基本基地（2–8 小時）
- 把收納做起來（至少分類箱；越晚做越痛）
- 先做一條最簡單的穩定產出（木材/食物/鐵）
- 準備下界：抗火/逃生/封門/路標

### 你已經去過下界（中期）
- 目標改成「打通終界」並拿到能提升效率/移動的戰利品
- 開始挑 1–2 個主力模組（科技/魔法都行），把它們做成你基地的核心

### 你開始看到奇點/大量壓縮需求（後期）
- 不要硬肝：把需求拆成「來源→加工→輸送→存放」並做成產線
- 先把物流與收納升級到能支援大量材料（否則你會一直找材料）

## 資源採集／量產方法（通用，配方以 JEI/REI）
- **木材**：樹場（半自動/自動）→ 集中輸送到收納/抽屜
- **鐵/金/銅等金屬**：
  - 早期：礦挖＋熔爐
  - 中期：礦物處理倍化（任一科技線）或資源作物/蜜蜂
- **紅石/青金/鑽石等稀有礦**：
  - 探礦＋幸運（或等效）
  - 後期改走資源作物/蜜蜂/維度採集
- **怪物掉落物**：安全刷怪場/掉落生成模組（如 Hostile Networks 類）
- **Cobblemon 消耗品（球類/治療/釣魚）**：先做「小規模手作補給」，中期再上自動化（杏果/金屬/加工）

---

## 常見卡關
- **任務很多不知道先做哪個**：回到這篇的 Phase 清單，先追里程碑（收納→維度→量產→星星）
- **材料太分散、一直找不到**：先升級收納與分類（抽屜/數位收納），再談產線
- **卡在需要大量資源**：你的下一步幾乎一定是「做農場/自動化」，不是繼續挖礦
- **探索找不到結構/生態域**：優先做指南針類工具（Nature’s/Explorer’s Compass）＋地圖路標
