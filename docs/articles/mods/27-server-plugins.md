# 伺服器加裝插件/模組清單（本服額外功能）

有些功能不是整合包「客戶端」自帶，而是伺服器額外加裝的插件/模組。

這頁是給玩家看的：你知道 **本服多了哪些功能**、大概怎麼用、遇到問題要去哪裡查。

---

## Cobblemon Challenge（玩家對戰指令）
**用途**：讓玩家可以用指令快速發起 Cobblemon 對戰（單打/雙打、幾打幾、是否隊伍預覽、等級統一）。

**常用指令**（簡表）：
- `/challenge <對方ID> level 50 nopreview`
- `/challenge <對方ID> 1v1` ～ `6v6`
- `/challengedouble <對方ID> 2v2` ～ `6v6`

**參考連結**：
- Modrinth：<https://modrinth.com/mod/cobblemon-challenge>
- 本站教學：<https://dubistowch.github.io/atm/index.html?guide=26-cobblemon-challenge>
