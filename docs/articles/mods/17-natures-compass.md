# Nature’s Compass 新手教學與玩法筆記

![模組圖示](https://cdn.modrinth.com/data/fPetb5Kh/9b0d54f2a8d9e49a10f5bcb8004bc757f55bb013.png)

## 簡介
Nature’s Compass 是用來找「生態域（Biome）」的工具：你選定目標生態域後，它會搜尋最近的位置並指路。

在整合包世界生成很複雜、或任務要求你去特定生態域時，它能省下大量「靠運氣跑圖」的時間。

## 模組特色
- 介面中選生態域 → 搜尋 → 指向最近的該生態域。
- 通常可調搜尋半徑（以及是否跨維度，依整合包）。
- 對有大量生態域模組（如各種地形/生態擴充）的整合包特別實用。

## 按鍵操作
Nature’s Compass 主要是手持右鍵開介面，不一定有固定鍵位。

- 打開「選項 → 控制設定」，搜尋：
  - `Nature`
  - `Compass`
  - `Biome`

若沒有鍵位也正常：代表主要操作都在物品右鍵介面。

## 遊戲內設定與調整
1) **在羅盤介面內用「關鍵字」找生態域**
- 很多整合包的生態域名稱是英文，且可能很長。
- 找不到時：用部分關鍵字即可，例如輸入 `cherry`、`mushroom`、`swamp`。

2) **搜尋半徑：先小後大**
- 半徑太小：找不到
- 半徑太大：搜尋時間長、伺服器可能更吃資源
- 建議做法：先用中小半徑測，找不到再逐步加大。

3) **找到後立刻做記錄**
- 建議你到達目標生態域邊界後：
  - 用地圖模組路標（JourneyMap）標記
  - 或放一個傳送點（Waystones 等）

4) **維度概念（避免白跑）**
- 多數情況：主世界生態域請在主世界找。
- 若整合包允許跨維度搜尋，以整合包規則與介面提示為準。

## 如何獲得模組道具
- 主要就是 Nature’s Compass 本體。
- 打開 JEI/REI 搜 `Nature's Compass` 看配方（多半早期就能做）。

## 好玩的點在哪
- **基地選址自由**：想住哪個生態域不靠運氣。
- **任務省時間**：要找沼澤、叢林、蘑菇島這類常常很遠的生態域時特別有感。
- **探索更有計畫**：找到後設路標/傳送點，你的世界會變成可回訪的旅行網路。

## 安裝與前置（可略讀）
- Nature’s Compass 多為單人/客戶端友善工具；伺服器可能會限制搜尋頻率或半徑以保護效能。
- 若你是管理員：搜尋半徑與冷卻等通常可在設定檔調整；修改前先備份。
