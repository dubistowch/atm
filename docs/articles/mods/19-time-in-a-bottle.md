# Time in a Bottle（瓶中時間）指南

> 目標：把你「站著不動的時間」存起來，之後用來加速機器/方塊運作，讓產線提速。

## 核心概念

Time in a Bottle 的玩法很直覺：

- 你在世界中活動時，瓶子會**累積時間**（像電池充電）。
- 之後你可以對某個方塊使用，讓它在一段時間內**加速 Tick**（更快運作）。

這特別適合：

- 早期沒有完整自動化、但想加速第一批資源
- 中期想把瓶頸產線（例如種植、加工、充能）短時間衝量

## 製作與取得

- 通常是偏早期可做的工具。
- 用 JEI 搜尋 `Time in a Bottle` 查看配方。

## 使用教學

1. 把 **Time in a Bottle** 放在身上（通常放快捷欄或背包即可累積）。
2. 當你有要加速的目標：
   - 手持瓶子
   - 對著目標方塊**右鍵**
3. 會出現加速倍率/剩餘時間等提示（顯示方式依版本）。

> 提醒：不同整合包可能限制可加速的方塊類型；有些會禁用對特定機器的加速。

## 適合加速的目標（高 CP 值）

- **作物/樹苗/甘蔗**：加速農作物成長、快速補貨交易物資。
- **蜜蜂/蜂巢相關**：加速產蜜（依整合包與方塊是否允許）。
- **熔爐/高爐/煙燻爐**：前期缺燃料/缺時間時很好用。
- **初期機器（粉碎/冶煉/洗礦）**：短時間把礦處理完。
- **發電/充能流程中的瓶頸機器**：例如某個加工速度太慢。

## 不太建議的用法

- 對「已經很快」或「已經滿載」的機器使用，提升有限。
- 對會造成大量輸出、但你倉儲/輸送跟不上的產線：
  - 可能造成掉落物滿地、容器爆滿、管線堵塞。

## 實戰套路：把瓶子當成「早期加速器」

1. 前期先存一段時間（邊探險、挖礦、蓋家）。
2. 回家後把時間砸在：
   - 甘蔗/小麥 → 交易紙/麵包
   - 熔爐把礦全燒完
   - 一兩台關鍵機器把材料加工到位
3. 產出第一批關鍵資源後，再轉向完整自動化。

## 常見問題

### Q1：為什麼我對某方塊右鍵沒反應？

- 該方塊可能被整合包禁止加速。
- 你可能需要對「方塊本體」而不是管線/外殼點擊。
- 可能是權限/伺服設定限制。

### Q2：時間怎麼累積得更快？

- 基本上是「你在線的時間」。
- 若整合包有額外規則（例如放在背包/快捷欄加成），以該包說明為準。
