# Time in a Bottle 新手教學與玩法筆記

## 簡介
Time in a Bottle（瓶中時間）是一個很直覺的加速道具：你在線上活動時它會累積「時間」，之後你可以對某個方塊使用，讓它在一段時間內加速運作。

它很適合用在「前中期還沒做完整自動化，但想短時間衝出一批材料」的情境。

## 模組特色
- 在線時自動累積可用時間（像充電）。
- 對方塊使用後可加速 tick（顯示方式依版本/整合包）。
- 整合包可能限制可加速的方塊種類（避免過於破壞平衡）。

## 按鍵操作
通常不需要鍵位：手持道具對方塊右鍵操作為主。

- 打開「選項 → 控制設定」，搜尋：
  - `Time in a Bottle`
  - `Time`

若沒有鍵位也正常。

## 遊戲內設定與調整
1) **加速前先確保「輸入/輸出不會爆」**
- 很多機器加速後問題不是產量高，而是：
  - 輸出容器太小 → 滿了停機
  - 管線太慢 → 堵住
  - 原料不夠 → 加速也白加速

2) **優先加速「你當下的瓶頸」**
- 你會最有感的通常是：
  - 作物（甘蔗/小麥等）→ 交易或加工原料
  - 熔爐/加工機 → 把一批礦快速處理完
  - 某條產線中最慢的一台機器

3) **整合包限制要注意**
- 如果你對某方塊右鍵沒反應：
  - 可能該方塊被整合包禁止加速
  - 或你點到外殼/管線而不是目標方塊本體

## 如何獲得模組道具
- 主要就是 Time in a Bottle 本體（配方以 JEI/REI 為準）。
- 通常偏早期就能做。

## 好玩的點在哪
- **前期加速器的爽感**：挖礦回家把瓶子時間砸在加工線，馬上得到第一批關鍵材料。
- **救急很好用**：臨時缺一批材料或要趕任務進度時特別有感。
- **搭配交易/農場很舒服**：加速作物 → 交易物資更穩 → 經濟/附魔推進更快。
