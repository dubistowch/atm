# Explorer’s Compass（探險羅盤）指南

> 目標：快速定位世界結構（Structure），例如村莊、海底神殿、掠奪者前哨站、要塞等。

## 你為什麼需要它

在整合包世界生成中，結構可能：

- 距離很遠
- 被新生態域/地形覆蓋很難肉眼找到
- 任務需要你「找到一次」才能繼續進度

Explorer’s Compass 提供一個簡單流程：**選結構 → 搜尋 → 指路**。

## 製作與取得

- 通常屬於**早期～中期**工具。
- 配方以 JEI 搜尋 `Explorer's Compass` 為準。

## 使用教學

1. 手持 **Explorer’s Compass**。
2. **右鍵**開啟介面。
3. 從清單選擇你要找的結構（或用搜尋框）。常見例子：
   - `Village`（村莊）
   - `Pillager Outpost`（掠奪者前哨站）
   - `Ocean Monument`（海底神殿）
   - `Stronghold`（要塞）
   - `Mansion`（林地府邸）
   - `Bastion Remnant`（堡壘遺跡，下界）
   - `Nether Fortress`（地獄堡壘，下界）
4. 開始搜尋後，羅盤會指向最近的該結構並顯示距離。

> 注意：結構是否出現在清單，取決於你整合包的結構模組與維度。

## 推薦用法（少走冤枉路）

- **先設定合理半徑**：
  - 太小：找不到
  - 太大：搜尋時間長
  - 建議逐步加大
- **到對應維度再搜**：
  - 例如找 `Nether Fortress` 請在下界使用。
- **找到後立刻記座標/放路標**：
  - 結構常需要多次回訪（刷凋零骷髏、找終界門等）。

## 實戰清單（ATM 常見任務/素材）

- **要塞 Stronghold**：找終界門、開終界。
- **海底神殿 Ocean Monument**：海晶石、海綿。
- **掠奪者前哨站 Outpost**：不祥之兆、掠奪相關掉落。
- **林地府邸 Mansion**：圖騰、稀有戰利品（依版本）。
- **地獄堡壘 Nether Fortress**：烈焰棒、地獄疣。
- **堡壘遺跡 Bastion**：豬布林、金裝備、部分模組進度。

## 常見問題

### Q1：為什麼找不到某個結構？

- 可能原因：
  - 該結構在這個世界生成配置中不存在
  - 你不在正確維度
  - 搜尋半徑太小
  - 與其他結構模組衝突（少見）

### Q2：搜尋很卡/很久？

- 降低搜尋半徑。
- 避免在伺服尖峰時段大量搜尋。
- 若整合包有設定可限制搜尋頻率，依規則使用。
