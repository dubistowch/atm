# Explorer’s Compass 新手教學與玩法筆記

## 簡介
Explorer’s Compass 是用來找「世界結構（Structure）」的工具：你在介面選擇村莊、要塞、海底神殿、下界堡壘等結構後，它會搜尋最近的位置並指路。

在整合包結構很多、地形很誇張、或任務要求你「找到一次」的時候，它能避免你無止境地亂跑。

![Explorer’s Compass 結構搜尋介面示意（截圖）](https://cdn.modrinth.com/data/RV1qfVQ8/images/4e701b15485b8739871047de1cf81f75920d3b7c_350.webp)

*圖：在介面中選定目標結構後開始搜尋，羅盤會指向最近的位置。重點是「先用關鍵字縮小清單」，並視伺服器效能調整搜尋半徑。*

## 模組特色
- 選結構 → 搜尋 → 指向最近的該結構。
- 通常可調搜尋半徑（依整合包）。
- 結構清單會依世界生成與整合包模組而不同。

## 按鍵操作
Explorer’s Compass 主要是手持右鍵開介面，不一定有固定鍵位。

- 打開「選項 → 控制設定」，搜尋：
  - `Explorer`
  - `Compass`
  - `Structure`

## 遊戲內設定與調整
1) **到正確維度再找**
- 找下界堡壘就去下界用；找主世界村莊就在主世界用。
- 這能避免你在錯誤維度「永遠找不到」。

2) **搜尋半徑：逐步加大**
- 半徑太小找不到很正常；逐步加大直到找到。
- 在伺服器上不要瘋狂重複搜尋，以免造成負擔。

3) **找到後立刻做路標/傳送點**
- 結構通常要回訪（刷烈焰棒、找終界門等）。
- 用 JourneyMap 路標或 Waystones 設點，之後就不用再找第二次。

4) **如果清單沒有你要的結構**
- 可能原因：
  - 這包世界生成沒有該結構
  - 結構被其他模組取代/改名
- 你可以嘗試用關鍵字搜尋相近名稱，或查整合包任務/百科。

## 如何獲得模組道具
- 主要就是 Explorer’s Compass 本體。
- 打開 JEI/REI 搜 `Explorer's Compass` 看配方（多半早中期即可取得）。

## 好玩的點在哪
- **任務推進更順**：要塞、神殿、府邸、堡壘這些「一找就很久」的目標變得可控。
- **探索更像冒險**：你知道方向與距離，路上就能規劃補給與回程。
- **和 Nature’s Compass 是一套**：一個找生態域、一個找結構，基本解決 80% 迷路問題。
