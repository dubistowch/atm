# Draconic Evolution 新手教學與玩法筆記

## 簡介
Draconic Evolution（DE）是偏後期的「超高階裝備與能量」模組，核心素材通常是 Draconium（龍能/龍晶相關礦物與材料），一路推進到更高階材料，最後做出非常強力的工具、武器、護甲與能量相關方塊。

很多整合包會把它放在中後期目標線：當你覺得探索/打王/跑材料開始吃力，DE 往往會帶來「戰力質變」。

## 模組特色
- 後期科技線清晰：核心礦物 → 高階材料 → 頂級裝備/能源設施。
- 裝備通常可升級/可客製化（依版本/整合包）。
- 提供高階能量儲存/能量相關方塊（多以 RF/FE 生態呈現）。
- 做出關鍵裝備後，移動、戰鬥與採集效率會大幅提升。

## 按鍵操作
DE 的快捷鍵很容易因版本與整合包而不同（也常跟其他 HUD/裝備模組衝突），**請以遊戲內為準**。

- 打開「選項 → 控制設定」，搜尋：
  - `Draconic`
  - `Draconic Evolution`
  - `Armor`（若有裝甲能力切換）
  - `Mode`

## 遊戲內設定與調整
1) **先把鍵位衝突整理好（後期體驗差很多）**
- 後期整合包功能鍵很多，DE 的模式/能力切換如果衝突會非常痛苦。
- 建議：把「切換模式/能力」集中到一組你記得住的按鍵，並在 Controls 裡測一次。

2) **用 JEI/REI 把材料鏈看懂再開工**
- DE 常見卡點不是「不知道怎麼做」，而是「漏做了一個中間材料」。
- 做法：從你想要的裝備往回追材料，列出 3～5 個必做中間件，照清單做。

3) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Draconic Evolution → `Config/設定`（若有）
  - 找不到就先不用管；玩家通常照整合包進度走即可。

> 補充（只留短註）：裝備強度、耗能、礦物生成等多在 config/資料檔；一般玩家不建議自行改動。

## 如何獲得模組道具
以生存進度整理（配方與名稱以 JEI/REI 與整合包為準）：

- **先穩定取得 Draconium 類資源**
  - 來源可能是地下礦物、特定維度，或整合包自訂產出方式。

- **把基礎材料做成可量產**
  - 先把礦物處理、熔煉/加工線穩定，後期材料需求會暴增。

- **再衝高階材料與裝備**
  - 覺醒/高階材料（依版本）通常是後期門檻。
  - 裝備與能量方塊往往需要大量能源與中間件。

代表性目標（方向列舉）：
- Draconium 相關礦物/錠/粉
- 高階材料（覺醒類材料，依版本）
- 高階工具/武器/護甲
- 高階能量儲存/能源方塊

## 好玩的點在哪
- **後期「變強」的成就感非常明確**：做出關鍵裝備後，你會感覺整個世界難度下降一個檔。
- **把它當作整合包的終局目標之一**：材料鏈長、需求大，適合當長線工程。
- **提醒：別太早把難度玩壞**：如果你還想保留探索/打王的挑戰感，可以先訂好「做到哪一階就停」。
