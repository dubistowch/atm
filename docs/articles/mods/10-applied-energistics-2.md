# Applied Energistics 2 教學與玩法筆記

## 簡介（這個模組在整合包的角色）
Applied Energistics 2（常簡稱 AE2）是整合包中最常見的「數位化倉儲與自動合成」核心模組之一。它把物品儲存、分類、搜索、物流與自動合成整合成 **ME 網路**：你可以把大量物品「存進系統」、用終端機快速搜尋取用，並用自動合成把工廠流程做成一鍵下單。

## 模組特色（條列，含 5-10 點）
- **ME 網路**：以線路/通道/電力構成的數位倉儲與物流系統（細節依版本設定）。
- **終端機體驗**：快速搜尋、排序、模糊查找、取放物品，取代成排箱子。
- **自動合成**：用樣板/合成 CPU 把常用配方做成「下單→自動製作」。
- **儲存介質**：以儲存元件管理容量，方便升級與擴充。
- **與其他模組整合性高**：可作為全整合包的自動化中樞，串接加工機、產線與農場。
- **玩家指南完整**：官方提供版本化的 Players Guide，查資料很方便。

## 按鍵操作
AE2 本身多數操作在方塊/終端 GUI 中完成，**不一定有固定必備快捷鍵**；而且不同版本、不同附屬模組（例如各種終端/無線終端）可能新增按鍵。

- 請到：**選項 → 控制**
- 搜尋：`Applied Energistics`、`AE2` 或 `ME`
- 若找不到明確按鍵，通常表示該版本以 GUI 操作為主。

## 如何設定（含常見設定檔位置/遊戲內設定入口）
- 常見設定檔位置（依版本可能不同）：
  - `config/appliedenergistics2/`
  - `config/ae2-*.toml` 或 `config/appliedenergistics2-*.toml`
- 遊戲內設定入口：
  - 部分版本可在 **Mods** 清單內調整（若支援）。
- 你可能會想調的方向（以整合包允許的設定為準）：
  - 通道/網路限制、能耗、世界生成（石英）、UI 行為等。

## 如何獲得模組道具（生存模式取得路線：合成、掉落、交易、探索；舉 3-6 個代表性物品）
以下為典型入門到成形的代表性物品（配方依整合包為準）：
- **Certus Quartz（水晶石英/賽特斯石英）相關材料**：常用於早期零件與線路。
- **ME Controller（ME 控制器）**：ME 網路的大腦，系統規模一大通常需要它。
- **ME Drive（ME 磁碟機）**：放置儲存元件，形成可擴充的數位倉庫。
- **Storage Cell（儲存元件）**：決定你能存多少物品（不同等級容量不同）。
- **ME Terminal（終端機）**：玩家與網路互動的主要介面（取放、搜尋、合成下單）。
- **Pattern Provider / Molecular Assembler（樣板供應/分子組裝機）**：自動合成的核心模組群。

## 好玩的點在哪（給玩家的玩法建議與小技巧）
- 先做「倉儲」再做「自動合成」：早期先把雜物整理進 ME 系統就很爽；等資源穩定再擴到自動合成。
- 重要物品先做「緩衝」：農場/加工線輸出很大時，先用箱子或其他儲存做緩衝，再接入 ME，避免瞬間爆量導致供電或處理瓶頸。
- 善用官方指南：AE2 的系統性很強，照著 Players Guide 推進會少走很多彎路。
