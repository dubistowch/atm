# Apotheosis 新手教學與玩法筆記

## 簡介
Apotheosis 是「把原版系統做深、做大」的強化型模組：它不一定大量新增機器或新礦，而是針對附魔、鐵砧、刷怪磚、戰利品與世界怪物強度等機制，加入更高上限與更多層的養成。

在整合包中它常扮演「刷裝與角色成長」核心：你會更有動力去跑結構、刷戰利品、打造附魔與強化裝備。

## 模組特色
- 以多個子系統/模組區塊構成（依版本可分別調整）。
- **附魔系統擴充**：提高上限、加入新附魔/新邏輯（依版本）。
- **鐵砧強化**：部分版本允許更多操作或更高上限（依版本）。
- **刷怪磚玩法擴充**：刷怪磚可被更細緻地改造（整合包常會做平衡限制）。
- **戰利品與詞綴（Affix）/強敵**：探索回報更高，但也更危險（依版本）。

## 按鍵操作
Apotheosis 多為系統與介面操作，通常不依賴固定鍵位。

- 打開「選項 → 控制設定」，搜尋：
  - `Apotheosis`
  - `Affix`
  - `Spawner`

若沒有找到鍵位也正常：多數功能會出現在附魔台/鐵砧/刷怪磚/物品說明中。

## 遊戲內設定與調整
1) **附魔台：把它當成「角色成長台」而不只是附魔一次**
- 你會更常回到附魔台：刷更好的附魔組合、規劃功能型附魔（挖礦/戰鬥/移動）。
- 建議做法：
  - 先把經驗來源穩定（刷怪塔/經驗機器/交易等）
  - 再開始追求高品質附魔，而不是一拿到效率就停。

2) **物品詞綴/戰利品判讀（如果你常看到彩色詞綴）**
- 如果整合包有開啟 Affix 類戰利品：
  - 先學會看物品描述：哪些加成是你需要的、哪些只是看起來很猛。
  - 建議準備「拆解/回收」的處理方式，避免倉庫被垃圾裝塞爆（依整合包提供的回收手段）。

3) **刷怪磚：先小規模測試再擴大**
- Apotheosis 的刷怪磚系統常常很強，但整合包也最常限制它。
- 你可以先做一個小房間測試：生成速度、掉落量、伺服器負載是否可接受，再決定要不要擴建。

4) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Apotheosis → `Config/設定`（若有）
  - 找不到就先不用管；玩家通常只要理解「附魔/戰利品/刷怪磚」三件事就夠玩。

> 補充（只留短註）：平衡與上限（附魔、刷怪磚、戰利品）多在 config/資料檔；一般玩家不需要改檔。

## 如何獲得模組道具
Apotheosis 多數內容綁在「原版系統」上，取得方式也更像「去玩原版但更深」：

- **附魔與鐵砧相關**：
  - 照常做附魔台/鐵砧，但你會花更多時間在組合與升級。

- **探索與戰鬥**：
  - 多跑地牢、結構與高風險區域，通常更容易拿到好戰利品（依版本/整合包）。

- **刷怪磚與材料農場**：
  - 若整合包允許：刷怪磚系統可以變成材料與經驗的核心供應。

代表性內容（以方向列出）：
- 新/強化的附魔與附魔書
- 詞綴戰利品（若有）
- 刷怪磚改造與其配套材料（若整合包開放）

## 好玩的點在哪
- **探索更有回報**：你跑結構不是為了「可能有鑽石」，而是為了刷到真正能改變玩法的裝備。
- **附魔變成養成**：你會開始追求「挖礦套」「戰鬥套」「移動套」這種功能分工。
- **中後期的刷裝循環很耐玩**：打更強的怪 → 拿更好的裝 → 再去挑戰更難的內容。

## 安裝與前置（可略讀）
- Apotheosis 影響面很廣（附魔、戰利品、刷怪磚…），整合包通常會搭配腳本/資料包做平衡。
- 玩家端：確保模組版本一致即可；不要自行更動檔案以免和伺服器/整合包設定衝突。
