# Easy Villagers（簡易村民）指南

> 目標：把繁瑣的「拉村民、蓋工作站、分類、補貨」流程，濃縮成可攜帶、可擴充、好管理的一套方塊化系統。

## 你會用到它的時機

- **前期**：想快速取得穩定附魔書、玻璃、末影珍珠、紅石等交易資源，但不想蓋大型村民交易所。
- **中期**：需要把村民集中管理（補貨、繁殖、治療、升級）並降低走失/暴斃風險。
- **多人/伺服**：把村民收納在小面積、低 Lag 的配置，減少實體碰撞與路徑計算。

## 核心概念（這模組在做什麼）

Easy Villagers 的重點是：

1. **把「村民」變成可以裝進物品的形式**（通常透過特定物品/操作收納）。
2. 用各種**功能方塊**取代傳統交易所需要的大量空間與 AI 行為：
   - 交易
   - 繁殖
   - 自動補貨/工作
   - 治療/轉職管理（依整合內容而定）

> 實際方塊名稱/配方會依你玩的整合包版本略有不同；以下以「功能與配置方式」為主，照著做通常都能落地。

## 常見方塊與用途

### 1) Villager Box / 收納村民的容器

- 功能：把村民「打包」成可搬運的狀態，之後可放入交易/繁殖等方塊。
- 用法要點：
  - **先確定村民職業與交易**（尤其是附魔書）。
  - 打包前先讓他**補貨一次**，避免剛搬家就卡補貨。

### 2) Trading Station / 交易站（或類似名稱）

- 功能：把一位村民放進去後，你可以直接對方塊介面交易，不需要村民走位。
- 實戰技巧：
  - **一站一村民**：用告示牌/命名標籤標記（例如「Mending」「Silk Touch」）。
  - 交易站旁放箱子/抽屜：集中收納綠寶石與交易產物。
  - 若整合包有「自動補貨/自動交易」：先確認是否需要紅石訊號、或有交易冷卻。

### 3) Auto Trader / 自動交易器（若有）

- 功能：消耗輸入物自動完成交易，輸出到容器。
- 注意事項：
  - **綠寶石供應**是瓶頸：建議搭配其他模組的農場（甘蔗→紙、南瓜/西瓜、石匠等）。
  - 交易會受村民補貨限制；若卡住，先看是否需要床/工作站條件（依整合包設定）。

### 4) Breeder / 繁殖器（若有）

- 功能：把兩位村民放進去，提供食物與床位條件後自動繁殖。
- 配置要點：
  - 食物：通常是麵包、胡蘿蔔、馬鈴薯、甜菜根等。
  - 產物：會產出「新村民」的收納形態或直接輸出到容器（視版本而定）。

## 推薦流程（從 0 到 1）

1. **先拿到第一批好交易**
   - 目標：修補 Mending、耐久 III、效率 V、精準採集、時運等。
2. **把理想村民打包**
   - 先確認交易固定，再收納。
3. **建立一個「交易牆」**
   - 每個村民一個交易站，統一接到集中倉儲。
4. **補上綠寶石來源**
   - 例如紙、玻璃、石頭、鐵工具等交易線。
5. （可選）**做自動交易**
   - 讓常用耗材（玻璃、末影珍珠、紅石、青金石、瓶子等）被動累積。

## 省心配置建議

- **村民安全**：把所有站點放在封閉室內，避免雷擊/殭屍/巡邏隊。
- **分類命名**：
  - 「附魔書」按用途：工具/防具/武器/通用。
  - 「材料」按來源：石匠/牧師/工具匠等。
- **備援**：核心附魔（如 Mending）建議至少保留 2 位村民，以防交易被改動或誤操作。

## 常見問題

### Q1：村民不補貨/交易灰掉？

- 先確認：
  - 村民是否有對應工作站條件（某些整合包仍要求）。
  - 是否到補貨時間/已到每日上限。
  - 是否被其它自動化鎖住（例如輸入不足）。

### Q2：打包後職業會重置嗎？

- 一般來說**交易鎖定後不會重置**，但建議：
  - 先交易一次把職業與交易固定。
  - 避免在未鎖定前搬運。

## 搭配推薦（ATM 常見搭法）

- **FTB Ultimine**：挖出村民基地/交易室空間更快（見另一篇）。
- **Jade**：看方塊資訊、容器內容、村民狀態更直覺（見另一篇）。
- **Time in a Bottle**：加速作物/資源生產，支援交易消耗品供應（見另一篇）。
