# Pocket Storage 新手教學與玩法筆記

![模組圖示](https://minecraft.wiki/images/Chest_JE2_BE2.png)

## 簡介
Pocket Storage（口袋收納/口袋空間類模組）通常會給你一個可攜帶的「私人小空間」：外出探險時背包滿了，你可以打開口袋空間進去整理、放箱子、當臨時補給站。

把它想成「隨身倉庫」或「行動基地雛形」。

## 模組特色
- 提供可進入的口袋空間/小房間（常與某個道具或入口方塊綁定）。
- 口袋內可放置容器、工作台等（整合包可能限制可放的方塊）。
- 讓探索不必頻繁回家卸貨。

## 按鍵操作
多數情況是手持道具右鍵或使用入口方塊，不一定有固定鍵位。

- 打開「選項 → 控制設定」，搜尋：
  - `Pocket`
  - `Storage`

## 遊戲內設定與調整
1) **先做一個「卸貨型口袋」就很夠用**
- 口袋空間內建議先放：
  - 1～2 個大容量箱子/抽屜（分類礦石與雜物）
  - 工作台
  - （可選）熔爐/簡易加工
  - （可選）垃圾桶/銷毀桶（清圓石、泥土）

2) **安全第一：不要在危險場合打開**
- 例如戰鬥中、岩漿旁、虛空邊緣。
- 先找安全點再開，避免出入口位置出事。

3) **重要物資不要只放一份**
- 如果整合包有死亡掉落/物品遺失風險：
  - 核心道具建議留備份在基地。

4) **共享/權限（若整合包有）**
- 口袋空間是否可共享、如何設定權限，依版本差異很大。
- 最快做法：看物品 tooltip、或查整合包說明。

## 如何獲得模組道具
- 用 JEI/REI 搜：`Pocket Storage` / `Pocket` / `Storage` 找到入口物品或方塊。
- 多數玩法圍繞「入口道具」展開。

## 好玩的點在哪
- **探索節奏超順**：不再因為背包滿就被迫回家。
- **前線基地感**：地牢門口開口袋整理、補給，再繼續深入。
- **搭配連鎖挖掘與背包模組更強**：大量挖掘 → 立刻進口袋卸貨分類。
