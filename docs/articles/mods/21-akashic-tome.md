# Akashic Tome 新手教學與玩法筆記

![模組圖示](https://cdn.modrinth.com/data/JBthPdnp/f31b97209d7973f2975c9162d6bf20a2f8488837_96.webp)

## 簡介
Akashic Tome（阿卡西之書）解決一個很現實的問題：整合包每個模組都給你一本指南書，背包很快被書塞爆。

它的做法是把多個模組的指南「合併」到同一本書裡，你只要帶一本 Tome，就能切換閱讀不同模組的手冊內容。

## 模組特色
- 合併多本模組指南書，節省背包空間。
- 打開後可切換到不同模組手冊（依支援度）。
- 非常適合和任務書/百科系統一起用。

## 按鍵操作
通常不需要鍵位：手持右鍵打開書本介面。

- 打開「選項 → 控制設定」，搜尋：
  - `Akashic`
  - `Tome`

若沒有鍵位也正常。

## 遊戲內設定與調整
1) **新手最推薦的習慣：拿到指南書就立刻合併**
- 很多整合包會透過任務或合成給你新的 Guide Book。
- 你可以把它當成「收到就歸檔」，避免散落在箱子裡找不到。

2) **如果有些書無法合併**
- 屬正常現象：不是每個模組的指南書都支援 Akashic Tome。
- 也可能是整合包版本/設定關閉了整合。

3) **把 Tome 放在快捷欄**
- 看到新機器、新材料卡關時：先翻指南書，通常比上網找更快（而且符合整合包改配方後的內容）。

## 如何獲得模組道具
- 主要就是 Akashic Tome 本體（配方以 JEI/REI 為準，通常很便宜）。
- 合併方式常見是：把 Tome 與目標指南書放在合成格/工作台（依整合包設定）。

## 好玩的點在哪
- **背包清爽**：一本書打包整包手冊。
- **學習速度變快**：遇到模組內容不熟，隨手翻就有。
- **很適合新手**：整合包資訊量大，Tome 等於把說明集中管理。
