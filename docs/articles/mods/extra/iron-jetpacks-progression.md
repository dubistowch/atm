# Iron Jetpacks：噴射背包製作與升級路線教學（材料階段、充電、常見升級）

> 本篇目標：讓你在整合包裡用「最少走彎路」的方式，從第一個能飛的背包一路升級到中後期，並且把充電與常用升級搞懂。
>
> 按鍵提醒：不同整合包會改鍵位。本文**不猜按鍵**；請到遊戲內 `Options → Controls` 搜尋 **"Jetpack"** 或 **"Iron Jetpacks"** 查看你實際綁定。

---

## 你會用到的模組／前置

- **Iron Jetpacks**（本體）
- （依整合包而定）任一種 FE 能量系統：
  - **Mekanism / Thermal / EnderIO / Powah / 任何能產生 FE 的模組**（整合包不同）
- （建議）**JEI/REI**：查看每一階 jetpack 的配方、以及升級（Upgrades）配方

參考來源：Iron Jetpacks 官方 GitHub 介紹頁面（提供官方站點/文件/Modrinth 等連結）。

---

## 目標與準備

### 目標
1. 做出第一個 Jetpack（通常是低階：Stone/Iron 一類，依整合包配方）
2. 學會「怎麼充電、怎麼維持能源供應」
3. 用升級（Upgrades）讓飛行體驗從「能飛」變成「好用」
4. 形成升級路線：低階 → 中階 → 高階

### 準備
- 一套基礎裝備（掉落保護看你伺服器規則）
- 一個穩定的 FE 發電方式（哪怕很小也行）
- 一個能充電的設備（整合包通常會提供）：
  - 例如「充電台/充電器/能量儲存方塊」等（名稱依整合包）

## 步驟教學（照做版）

### 步驟 1：用 JEI 找到「你的第一階」Jetpack
1. 打開 JEI/REI 搜尋 `jetpack`。
2. 你會看到多個等級（tiers）。不同整合包可能有不同階段命名與材料要求。
3. 選擇你目前做得出來的最低階，先把它做出來。

> 心法：先拿到「能飛」的能力，再回頭優化。早期噴射背包最大的價值是移動與探勘效率。

### 步驟 2：學會正確穿戴與基本設定
1. Jetpack 通常穿在「胸甲」欄位或對應的裝備槽（依整合包可能有 Curios/Trinkets 槽）。
2. 進 Controls 搜尋 Jetpack：
   - 找到「啟動/關閉」「模式切換」「加速/懸停」等功能（具體名稱以你版本為準）。
3. 先在家門口低空試飛：
   - 確認你會「起飛、降落、停在空中（若有）」

### 步驟 3：建立充電流程（沒有電 = 只有背包外觀）
1. 看 Jetpack 介面/提示，確認它的能源類型是 **FE**。
2. 放一個你整合包提供的充電設備（例如充電器方塊）。
3. 把 Jetpack 放進去充電（或用能量道具給它充）。
4. 充電不足時會出現：
   - 飛不動/推力不足
   - 起飛一下就掉下來

> 你要追求的是：回家就「順手丟進充電」成為習慣；中後期才會做自動充電/隨身供電。

### 步驟 4：升級路線怎麼選（不硬背材料，按你的資源走）
常見路線（概念版）：

- **低階（能飛）**：材料以石材/鐵/紅石等常見資源為主
- **中階（好用）**：開始吃大量金屬、合金、能量元件
- **高階（爽飛）**：需要更高階合金、稀有材料、更多能源吞吐

做法：
1. 你每升一階，都先確認兩件事：
   - 你的發電是否跟得上（充得滿/用得久）
   - 你是否已經有必要的「升級卡/模組」讓手感改善
2. 若發電跟不上：
   - 先升級發電/電池/線材，再升 Jetpack

### 步驟 5：常見升級（Upgrades）與實戰建議
不同版本的 upgrade 名稱與效果可能有差，但玩法上通常會遇到這些方向：

- **效率/耗電改善**：同樣電量飛更久（很適合探索）
- **推力/速度提升**：起飛更快、移動更爽（但耗電更兇）
- **懸停/穩定**：蓋房子、挖天花板、打 Boss 更好用
- **安全性**：掉落/緊急降落等（視整合包是否額外提供）

建議順序（新手穩定版）：
1. 先拿「穩定/懸停」類（如果你版本有）
2. 再拿「效率」
3. 最後堆「速度/推力」

## 常見坑與排查

1. **我做出來了但不能飛**
   - 檢查是否有電（FE）
   - 檢查是否有啟動（很多背包預設是關閉）
   - 檢查鍵位是否衝突（Controls 裡搜尋 Jetpack）

2. **飛一下就沒電**
   - 發電跟不上：先擴電力
   - 你可能裝了高耗電的速度/推力升級

3. **穿了胸甲就不能穿 Jetpack（或反過來）**
   - 可能是裝備槽衝突（胸甲欄 vs Curios）
   - 查整合包是否有提供可共存的配件/升級，或有「背包穿戴槽」設定

4. **我找不到升級怎麼做**
   - JEI 搜尋 `upgrade` 或 `jetpack`
   - 以你整合包實際配方為準（很多包會改）
