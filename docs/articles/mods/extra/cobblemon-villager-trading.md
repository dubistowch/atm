# Cobblemon：交易村民教學（取得／刷新／交易思路）

> 目標讀者：想把 Cobblemon 的道具（樹果、藥品、種子等）穩定量產、用綠寶石解決補給的生存玩家。
>
> 版本提醒：Cobblemon 會改動「新增職業／配方／交易內容」。本文以 **Cobblemon Wiki 的 Villager/Healing Machine 條目** 為基礎講玩法流程，**不提供固定交易表**（以免版本差異造成誤導）；實際交易內容請以你當下版本（或伺服器設定）為準。

---

## 你會用到的模組／前置

- **Cobblemon**（核心）
- （可選）**Easy Villagers**：把「養村民／刷交易／集中管理」變得更省空間、也比較安全（整合包有裝才適用）
- （建議）**JEI/REI**：查 Cobblemon 物品配方、以及村民交易需要的材料替代品
- 香草前置概念：村民職業、工作站、交易解鎖與補貨（restock）

---

## 目標與準備

### 目標
1. 取得可控的村民（自給自足的繁殖／轉職環境）
2. 做出 Cobblemon 的**新增職業村民**（目前 Wiki 提到的職業是 **Nurse**）
3. 用「不鎖死、可重刷」的方式，把你想要的交易刷出來
4. 把交易變成可量產的資源迴圈（綠寶石 ↔ Cobblemon 補給）

### 準備清單（生存向）
- 一個安全的小房間（2×3 也行）
- 床、工作站、食物（小麥／麵包／胡蘿蔔／馬鈴薯都行）
- 足夠的綠寶石來源（採礦、村民農夫交易、掠奪者塔等）
- **Healing Machine（治療機）**：Cobblemon 的治療方塊，並且可作為 Nurse 的工作站（Wiki 說明）

> Wiki 提到：Cobblemon 1.6 起 Healing Machine 可作為村民工作方塊，讓村民取得 Nurse 職業；村莊也可能生成帶有 Healing Machine 的 Pokécenter 結構（不同版本細節請看條目）。

## 步驟教學（照做版）

### 步驟 1：先做「可控村民」的基礎設施
1. 找到村莊，或用殭屍村民治療法取得村民。
2. 做一個**封閉工作間**：
   - 讓村民能站在工作站附近（1 格距離通常就能認領）
   - 避免村民跑走、或被怪物弄死
3. 先別急著交易：
   - **未交易過的村民**，移除工作站後通常能失業並重新刷職業／交易（香草機制；若伺服器有改規則以伺服器為準）。

### 步驟 2：放下 Healing Machine，刷出 Cobblemon 的 Nurse
1. 把村民關進工作間。
2. 在村民附近放下 **Healing Machine**。
3. 等待一小段時間（通常是白天工作時間）讓村民認領工作站並轉職。
4. 若沒有轉職：
   - 確認附近沒有其它工作站被他認走
   - 把其它工作站先拆掉，只留 Healing Machine
   - 讓村民能「看到／走到」工作站（路徑被擋住有時會認不到）

參考：Cobblemon Wiki 的 Villager、Healing Machine 條目。

### 步驟 3：用「不鎖交易」的方式重刷你要的商品
核心原則：**交易一旦做過，該村民通常會鎖定職業與交易池**。

刷法（最安全）：
1. 打開村民交易介面，看他的第一階段（Novice）有哪些交易。
2. 如果不滿意：
   - **先不要交易**
   - 拆掉 Healing Machine
   - 等村民變回無業（或刷新一輪）
   - 再放回 Healing Machine 讓他重新轉職
3. 重複直到你看到你想要的方向（例如你缺的樹果、藥材、種子等）。

> 注意：不同版本的交易池內容與數值會改；**不要照抄別人的交易表**。建議只把「玩法目標」定成：
> - 你想用綠寶石換哪一類 Cobblemon 物資
> - 你願不願意升級等級換更高階的交易

### 步驟 4：把它變成「能持續補貨」的交易站
1. 村民交易會有「庫存」與「補貨（restock）」概念：
   - 通常需要工作站存在、且在工作時間附近才會補貨
2. 你要做的配置：
   - Nurse 村民 + Healing Machine（工作站固定保留）
   - 旁邊放箱子／補給，放你要拿去換綠寶石的物資，或放綠寶石
3. 若整合包有 **Easy Villagers**：
   - 你可以考慮用它的「村民管理方塊／交易方塊」來集中化（依你整合包實際方塊與配方為準）
   - 好處：省空間、較安全、搬運方便

## 玩法思路（怎麼選交易比較賺）

- **先做綠寶石量產，再玩 Cobblemon 交易**：
  - 香草村民（農夫／圖書館員／石匠等）的循環常比你硬挖綠寶石穩定。
- **把 Nurse 當作「Cobblemon 消耗品補給站」**：
  - 冒險打道館／抓怪 → 回村莊用綠寶石補藥補樹果 → 繼續探險。
- **別急著升級**：
  - 看到想要的交易再開始交易升級；一旦交易過就很難回頭重刷。

## 常見坑與排查

1. **村民不轉職**
   - 先清掉附近其它工作站（他可能認了別的）
   - 確認是白天工作時段
   - 確認村民可以靠近工作站（路徑／高度差）

2. **拆了工作站他也不失業**
   - 你可能已經跟他交易過了（職業鎖定）
   - 伺服器可能有調整村民規則

3. **交易不補貨／庫存一直 0**
   - 工作站是否還在、村民是否仍能認領
   - 村民是否能進入工作狀態（有些封印方式會讓他卡住）

4. **Healing Machine 的配方跟你印象不一樣**
   - Cobblemon 版本曾調整配方（Wiki 有歷史變更）
   - 以你整合包的 JEI/REI 顯示為準
