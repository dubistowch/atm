# Cobblemon：考古與化石復活教學（可疑沙礫、刷子、化石、復活流程）

> 重點：
> - 化石主要來自 **Cobblemon 的化石結構（structures）** 裡的 **可疑沙礫（Suspicious Sand/Gravel）**
> - 用 **刷子（Brush）** 刷出 **Fossil**
> - 用 **Resurrection Machine（化石復活機，多方塊結構）** 把化石復活成寶可夢
>
> 配方提醒：Cobblemon/整合包可能調整配方與材料，本文流程教學會一直用「**JEI/REI 查配方**」作為準則。

---

## 你會用到的模組／前置

- **Cobblemon**（化石、復活機、化石結構）
- （建議）**JEI/REI**：查 Resurrection Machine 各方塊配方、以及需要的材料
- 香草：
  - **Brush（刷子）**
  - **Suspicious Sand / Suspicious Gravel（可疑沙礫）**
  - 考古玩法（刷方塊時會碎掉、要小心保護）

---

## 目標與準備

### 目標
1. 在野外找到帶有可疑沙礫的 Cobblemon 結構
2. 用刷子取得化石
3. 組裝 Resurrection Machine
4. 補足有機材料（Organic Materials）並完成復活

### 準備清單
- 刷子（Brush）：先準備 2 把以上，耐久不夠會很煩
- 冒險裝：食物、火把、方塊、（建議）水桶
- 回家要準備：
  - 做 Resurrection Machine 的材料（用 JEI 查）
  - 一堆「有機材料」當燃料／培養基（例如種子、植物類等；Wiki 有價值表）
  - 至少 1 顆你想收編用的精靈球（最後收取化石寶可夢用）

---

## 步驟教學（照做版）

### 步驟 1：先用「找結構」的思路找化石
1. 出門探索：海岸、河流、洞穴、各種地形都可能遇到 Cobblemon 結構。
2. 看到 Cobblemon 的結構遺跡（不同主題、不同材質）時：
   - 優先找看起來像「考古點」的區域：
     - 地上有一坨比較可疑的沙礫堆
     - 或結構內有可疑沙礫混在一般沙／礫石之中
3. 找到 **Suspicious Sand / Suspicious Gravel** 後：
   - 先用一般方塊（泥土／石頭）把周圍圍起來
   - 避免砂砾受重力掉落砸碎、或被你不小心挖掉

> Cobblemon Wiki 明確寫到：化石可由刷子刷可疑沙礫取得，位置在化石 structures 內。

### 步驟 2：刷出化石（Fossil）
1. 拿出 **Brush** 對著可疑沙礫長按使用。
2. 刷的過程中方塊會慢慢「被清理」，最後掉出戰利品。
3. 如果刷到的是 Cobblemon 的 **Fossil**：
   - 先別急著離開，附近通常不只一格可疑沙礫
   - 建議把整個結構都搜完再回家（效率比較好）

### 步驟 3：回家做 Resurrection Machine（三方塊多方塊結構）
根據 Cobblemon Wiki：Resurrection Machine 由三種方塊組成：
- **Fossil Analyzer**
- **Restoration Tank**
- **Data Monitor**

建造流程（新手版）：
1. 先用 JEI 查三個方塊怎麼做，全部做齊。
2. 在你家找一塊空地（建議室內 3×3，留高度空間）。
3. 依照 Wiki 的說明把三個方塊擺成正確結構。
4. 擺對時會有「通電/機械」的聲音提示（Wiki 描述）。

> 小技巧：先把三個方塊擺在地上，聽到提示再開始接紅石／漏斗等自動化。

### 步驟 4：放入化石，補滿有機材料（Organic Material Value 需要達到 64）
Wiki 重點（非常重要）：
- Restoration Tank 需要填入「有機材料」，**總價值達到 64** 才會開始復活流程。

實作建議：
1. 把化石放進 Fossil Analyzer（Wiki 提到可用漏斗插入）。
2. 往 Restoration Tank 投入有機材料：
   - 新手最穩的是：種子／植物類先大量農場化（小麥種子等）
   - 你也可以用 Cobblemon 的樹果、杏果等堆量（看你手邊資源）
3. 觀察 Tank 內部狀態（會有成長/培養的視覺效果）。

### 步驟 5：等待復活完成，選擇「放生」或「收編」
Cobblemon Wiki 描述：
- 復活需要 **12 分鐘**（完成後胚胎成熟）
- 完成後你可以：
  1. **打破 Tank** 讓寶可夢以野生形式出來
  2. 或者 **主手拿精靈球右鍵**，直接把復活出的寶可夢收進球並加入隊伍

新手建議：
- 第一次玩先用「主手拿精靈球右鍵」收編，最不容易出意外。
- 如果你想刷特性／性格／個體（看伺服器規則）：
  - 可以考慮放生再抓，但風險更高（跑走、被打死、被他人搶等）。

## 常見坑與排查

1. **我刷不到化石**
   - 你刷的可能不是可疑沙礫（外觀有差）
   - 你可能在一般遺跡而非 Cobblemon 的 fossil structures
   - 建議多找幾個結構；不要只靠一個點

2. **Resurrection Machine 擺了沒反應**
   - 三個方塊要「完整組裝」才能啟動
   - 請對照 Wiki 的 Resurrection Machine 條目圖示／描述

3. **放了化石也不開始復活**
   - Restoration Tank 的有機材料「總價值」可能不足 64
   - 你丟的是不算有機材料的東西（以 Wiki 的 Organic Material Values 或你版本為準）

4. **我想自動補有機材料但不知道丟哪些**
   - 先用「農場種子」當通用方案（大量且可自動化）
   - 再慢慢替換成更高價值/更好量產的物資
