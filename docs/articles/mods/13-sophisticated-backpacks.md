# Sophisticated Backpacks（可升級背包）新手攻略

Sophisticated Backpacks 的背包不是「多一排格子」而已：它更像一個**可升級的行動倉庫**。你可以加裝各種 Upgrade 來做到：
- 自動吸物 / 只吸你想要的東西
- 自動補貨（方塊、食物、火把等常用物品維持固定數量）
- 內建排序 / 快速搬運（背包 ↔ 箱子）
- 進階玩法：壓縮、工具/消耗品管理、與自動化互動（依版本/整合包配置）

> 版本差異提醒：不同 ATM 版本可能會調整配方、名稱或升級效果。
> **以你遊戲內 JEI/REI 與 Upgrade 說明 Tooltip 為準**。

---

## 0) 先把「開背包」按鍵設定好（不要猜按鍵）

1. 進遊戲後到：**Options → Controls**
2. 在搜尋框輸入：`backpack` / `sophisticated` / `open backpack`
3. 找到「開啟背包」相關綁定，改成你順手的鍵

> 本文不直接寫預設按鍵，避免不同整合包/版本造成誤導。

## 1) 前 30 分鐘新手流程（照做就有感）

這段目標：你不需要先理解所有升級，只要把「背包 → 整理 → 只吸想要的 → 常用物自動補」這套基本功做出來。

### Step 1：先做出第一個背包、穿在身上
- 打開 JEI/REI 搜 `Backpack`，先做**最基礎的背包**
- 確認你能用你綁定的按鍵打開它

> 建議：背包先當「隨身收納」，別急著亂塞所有東西，後面做篩選更省事。

### Step 2：學會 2 個最常用的操作：排序、快速搬運
打開背包 UI 後，先找到：
- **Sort（排序）**：一鍵把背包內物品整理好
- **搬運按鈕/快捷**：把你背包/箱子互相倒物（不同版本 UI 位置略不同）

### Step 3：先做「只吸礦物」或「不要吸垃圾」的基礎篩選
新手最常痛點是：背包吸了一堆泥土、種子、雜草，真正要的礦反而塞不下。

你可以做兩種流派（擇一先用就好）：

**流派 A：白名單（只吸想要的）**
- 適合：挖礦、地牢撿戰利品
- 作法：在背包的 Pickup / Filter 相關設定中，改成 Allowlist，填入：礦石、原礦、戰利品（你想撿的）

**流派 B：黑名單（不要吸垃圾）**
- 適合：探索時什麼都想撿，但不想撿泥土/石頭/種子
- 作法：改成 Denylist，填入：泥土、圓石、種子、腐肉…等你認定的垃圾

> ✅ 建議先用黑名單：更不容易「怎麼都撿不到」。
> 等你熟了再用白名單做精準撿拾。

### Step 4：做一個「補貨（Refill）」：火把或食物
如果你的版本有 Refill / Restock 類的功能（或 Upgrade）：
- 設定「讓快捷欄某格維持固定數量」
- 最常用：火把、食物、建材（圓石/木材）、箭矢

### Step 5（有餘力再做）：加第一個容量升級
優先做「增加容量」的那個（名稱依版本不同）。

## 3) 升級（Upgrades）怎麼選：從「最有感」開始

> 原則：**先容量 → 再撿取/補貨 → 再自動化/進階功能。**

### 3.1 容量相關（優先度：★★★★★）
你開始一次挖礦/探洞回來背包常滿，就該做。

### 3.2 撿取/補貨/整理（優先度：★★★★★）
新手推薦配置（探索通用）：
- 黑名單：泥土、圓石、種子、腐肉（依你需求調整）
- 補貨：火把、食物

### 3.3 壓縮/堆疊管理（優先度：★★★☆☆）
注意：自動壓縮可能讓你「需要原本型態」時不方便；先小量測試再開常駐。

### 3.4 進階互動（優先度：看流派）
可能包含工具/儲能/儲液/與方塊互動等；建議等你開始做自動化再研究。

## 4) 篩選（Filter）設定：最常見卡關一次講清楚

### 4.1 白名單 vs 黑名單
- Allowlist（白名單）：只允許列表內物品（漏加就撿不到）
- Denylist（黑名單）：禁止列表內物品（誤加就撿不到）

### 4.2 NBT/耐久/附魔：為什麼同名物品被當不同？
工具/裝備/書常常因為耐久、附魔、內部資料不同而被視為「不同物品」。

**排查建議**：先把 NBT/耐久比對調成「忽略」（如果你只是想撿同類物品）；要精準再打開比對。

---

## 5) 常見卡關（FAQ / 快速排查）

### Q1：我按了開背包沒反應
- 去 **Options → Controls** 搜 `backpack` / `sophisticated` 確認按鍵衝突或未綁定
- 確認背包是否真的在「可開啟」的位置（某些整合包有專用槽）

### Q2：背包一直吸垃圾，挖礦很痛苦
- 用黑名單先擋掉：泥土、圓石、種子、腐肉（依你世界調整）

### Q3：我想讓火把永遠維持 64，但它不會自動補
- 先確認你的版本是否需要某個 Refill/Restock Upgrade
- 再檢查：背包裡是否真的有火把可補、你設定的是哪一格快捷欄

### Q4：背包容量升級做了，但插上去沒變大
- 確認你插在「Upgrade 插槽」而不是一般格子
- 有些升級需要在設定頁啟用/調參（看 Tooltip）

### Q5：同樣物品堆不在一起，背包一下就滿
- 多半是 NBT/耐久/附魔不同造成
- 解法：調整 Filter 的 NBT/耐久比對，或把工具/裝備另外分類收納
