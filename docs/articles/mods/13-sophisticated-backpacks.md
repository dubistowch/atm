# Sophisticated Backpacks 新手教學與玩法筆記

## 簡介
Sophisticated Backpacks 是大型整合包裡最實用的 QoL 背包模組之一：它不只是「多一格空間」，而是讓背包變成可升級、可加裝功能的行動工作站。

你會在探索、下礦、跑地牢、蓋大工程時，明顯感覺到它讓整理時間大幅下降。

## 模組特色
- 背包可穿戴，也可放在地上當容器使用。
- 可染色/客製外觀。
- 容量可透過升級擴充。
- 可裝各式功能升級（依版本/整合包啟用），常見方向：
  - 自動撿取/吸取
  - 過濾與分類
  - 排序/搜尋
  - 工具/消耗品管理

## 按鍵操作
背包模組常有「開啟背包」「快捷互動」等鍵位，但整合包常重綁；**請以遊戲內為準**。

- 打開「選項 → 控制設定」，搜尋：
  - `Sophisticated`
  - `Backpack`
  - `Backpacks`

建議把「開背包」綁到你最常用且不衝突的鍵（避免跟地圖、語音、旁白、其他背包模組衝突）。

## 遊戲內設定與調整
1) **先把背包當成「固定配置」而不是垃圾袋**
- 最舒服的用法是：
  - 固定一格放食物
  - 固定一格放火把/照明
  - 固定一格放方塊（臨時搭橋）
  - 固定一格放備用工具
- 這樣你每次出門都像帶著工具箱，不用臨時找東西。

2) **先做過濾/整理，再做容量**
- 容量再大，沒有規則也會變垃圾場。
- 你可以先用「過濾/白名單/黑名單」類升級（名稱依版本）把：
  - 圓石、泥土、碎石等大量雜物導流到特定格
  - 稀有物品集中到另一區

3) **放置背包當前線倉庫**
- 下礦或跑遺跡時，把背包放地上就是臨時倉庫。
- 配合一個垃圾桶/抽屜/分類容器（依整合包）會更省心。

4) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Sophisticated Backpacks → `Config/設定`（若有）
  - 找不到就先不用管；玩家端多數情況不需要改。

> 補充（只留短註）：背包互動限制、升級行為等可能在 config；一般玩家以整合包預設為準。

## 如何獲得模組道具
通常以合成取得並逐步升級（配方以 JEI/REI 為準）：

- **背包本體**：越早做越舒服。
- **容量升級**：擴充欄位或提供更多空間。
- **功能升級**：吸取/過濾/整理/自動餵食等（依整合包）。

代表性物品（方向列舉）：
- Backpack（背包本體）
- Storage/Inventory Upgrade（容量升級）
- Pickup/Magnet/Filter 類 Upgrade（功能升級，名稱依版本）

## 好玩的點在哪
- **探索節奏變順**：背包滿了不用立刻回家，口袋裡就能分類。
- **減少整理時間**：從「回家整理半小時」變成「邊走邊整理」。
- **死亡回收更有條理**：常用物資集中在背包，比散在全身各格好找。
