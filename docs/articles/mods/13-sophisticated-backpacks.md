# Sophisticated Backpacks 教學與玩法筆記

## 簡介（這個模組在整合包的角色）
Sophisticated Backpacks 是偏「品質提升」的背包模組：提供可以穿戴、可放置在地上、可染色、可升級容量，並能安裝各式功能升級（例如自動撿取、過濾、餵食、工具整理等——實際可用升級依版本/整合包）。在大型整合包中，它常用來解決探索時背包空間不足、材料分類困難、雜物太多等問題。

## 模組特色（條列，含 5-10 點）
- 背包可穿戴，並可放置在世界中當作容器使用。
- 支援染色與多種配色組合，外觀可客製化。
- 可透過升級擴充背包容量（不同升級提供不同欄位/效果）。
- 有多種功能性升級，讓背包從「收納」變成「行動工作站」（依整合包啟用）。
- 通常提供搜尋、排序、過濾等便利功能，降低整理時間。
- 適合探索、刷怪、採集、建築等所有需要大量攜帶物資的玩法。

## 按鍵操作
背包模組常見有「打開背包」「快捷放置/互動」等按鍵，但鍵位會因版本與整合包調整。

- 請到：**選項 → 控制**
- 搜尋：`Sophisticated Backpacks`、`Backpack` 或 `Sophisticated`
- 把「開啟背包」設成你習慣的鍵位，並避免與 JourneyMap（常用 B）、其他背包模組、語音/旁白等鍵位衝突。

> 本篇不直接列出預設鍵位，避免因版本差異造成誤導。

## 如何設定（含常見設定檔位置/遊戲內設定入口）
- 常見設定檔位置（依版本可能不同）：
  - `config/sophisticatedbackpacks/` 或 `config/sophisticatedbackpacks-*.toml`
- 遊戲內設定入口：
  - 部分版本可在 **Mods** 清單中進入設定（若支援）。
- 常見可調方向（以實際版本為準）：
  - 升級行為、撿取/過濾規則、互動限制、背包放置行為等。

## 如何獲得模組道具（生存模式取得路線：合成、掉落、交易、探索；舉 3-6 個代表性物品）
一般以合成與升級為主：
- **背包本體（基礎背包）**：通常早期即可合成，作為行動儲存。
- **容量升級（Inventory/Storage Upgrade 類）**：擴大欄位或提升可用空間。
- **功能升級（Upgrade 類）**：例如過濾、吸取、餵食、工具/武器處理等（名稱依版本）。
- **外觀相關（染色/裝飾）**：透過染料等物品改變外觀。

> 具體升級種類很多，且整合包可能禁用部分升級或改配方，建議用 JEI/REI 直接查「upgrade」或背包物品頁籤。

## 好玩的點在哪（給玩家的玩法建議與小技巧）
- 探索必帶：把常用工具（火把、食物、方塊、弓箭等）固定放背包，死亡回收也更有條理。
- 先做「整理/過濾」再做「容量」：容量再大，沒規則也會變垃圾場；先建立分類習慣。
- 放置背包當臨時基地箱：下礦或跑遺跡時，背包放地上就是你的前線倉庫。
- 鍵位與 UI 衝突要整理：大型整合包常見 B、Shift+B、Ctrl+B 等衝突，早點整理最省事。
