# 進階背包（Sophisticated Backpacks）怎麼做、怎麼用：新手上手手冊

## 簡介
Sophisticated Backpacks 是大型整合包裡最實用的 QoL 背包模組之一：它不只是「多一格空間」，而是讓背包變成可升級、可加裝功能的行動工作站。

你會在探索、下礦、跑地牢、蓋大工程時，明顯感覺到它讓整理時間大幅下降。

## 模組特色
- 背包可穿戴，也可放在地上當容器使用。
- 可染色/客製外觀。
- 容量可透過升級擴充。
- 可裝各式功能升級（依版本/整合包啟用），常見方向：
  - 自動撿取/吸取
  - 過濾與分類
  - 排序/搜尋
  - 工具/消耗品管理

## 按鍵操作
背包模組常有「開啟背包」「快捷互動」等鍵位，但整合包常重綁；**請以遊戲內為準**。

- 打開「選項 → 控制設定」，搜尋：
  - `Sophisticated`
  - `Backpack`
  - `Backpacks`

建議把「開背包」綁到你最常用且不衝突的鍵（避免跟地圖、語音、旁白、其他背包模組衝突）。

## 遊戲內設定與調整
1) **先把背包當成「固定配置」而不是垃圾袋**
- 最舒服的用法是：
  - 固定一格放食物
  - 固定一格放火把/照明
  - 固定一格放方塊（臨時搭橋）
  - 固定一格放備用工具
- 這樣你每次出門都像帶著工具箱，不用臨時找東西。

2) **先做過濾/整理，再做容量**
- 容量再大，沒有規則也會變垃圾場。
- 你可以先用「過濾/白名單/黑名單」類升級（名稱依版本）把：
  - 圓石、泥土、碎石等大量雜物導流到特定格
  - 稀有物品集中到另一區

3) **放置背包當前線倉庫**
- 下礦或跑遺跡時，把背包放地上就是臨時倉庫。
- 配合一個垃圾桶/抽屜/分類容器（依整合包）會更省心。

4) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Sophisticated Backpacks → `Config/設定`（若有）
  - 找不到就先不用管；玩家端多數情況不需要改。

> 補充（只留短註）：背包互動限制、升級行為等可能在 config；一般玩家以整合包預設為準。

## 最短上手流程（照做就會用）
- [ ] 打開 JEI/REI（物品查詢）搜尋：`backpack`
- [ ] 先做「最基本的 Backpack（背包本體）」
- [ ] 做 1～2 個你最需要的升級（容量或排序/過濾）
- [ ] 把背包**穿在身上**或放在快捷欄，確認你能「打開背包介面」
- [ ] 出門前固定放：食物/火把/搭橋方塊/備用工具（別把它當垃圾袋）

## 如何獲得模組道具（不瞎猜配方版）
這包的配方可能有調整，所以我不直接寫「需要幾張皮革/幾個箱子」這種很容易寫錯的內容。

你用下面這個流程，一定找得到：

### 1) 找到背包本體的配方
1. 打開物品查詢（JEI/REI）
2. 搜尋 `backpack`
3. 點選 **Backpack（背包本體）**
4. 按「顯示配方」→ 照著做

> 小技巧：把背包配方「釘選」在畫面上（JEI 常見功能），邊挖材料邊看。

### 2) 找到升級（Upgrade）的取得方式
同樣在 JEI/REI 搜尋：
- `upgrade`
- `stack`
- `filter`
- `pickup`
- `magnet`
- `sort`

看你想要的功能，先做 1～2 個就好；升級做太多反而會讓背包變難整理。

## 怎麼使用（穿戴、開啟、裝升級）

### 1) 背包怎麼「穿在身上」？
常見有兩種情況（依整合包與前置模組而定）：
- **有背包/飾品欄位**：打開角色資訊（背包/飾品那頁）把背包拖進去。
- **沒有專用欄位**：就把背包放在快捷欄，用「開啟背包」鍵位打開。

如果你不確定入口：
- 先看背包物品的 tooltip（滑鼠移上去）有沒有寫「可裝備」或提示。
- 再去 **選項 → 控制設定** 搜：`Backpack` / `Sophisticated`。

### 2) 背包怎麼「打開」？
通常會有其中一種：
- 對著背包物品右鍵（手上拿著）
- 或按「開啟背包」的鍵位

> 你不用猜：**先去 Controls 搜關鍵字**，找到「Open Backpack / Backpack」相關動作，照它顯示的按鍵操作。

### 3) 升級要裝在哪裡？
背包介面裡通常會有「Upgrade/升級」區塊（或一個像插槽/模組化面板的地方）：
1. 打開背包
2. 找到 Upgrade/升級區
3. 把做好的 Upgrade 拖進去
4. 如果升級有設定（例如過濾清單），背包介面旁邊通常會出現齒輪/設定按鈕

### 4) 「放地上當容器」要怎麼放？
不同版本操作會不一樣，所以這裡只給**不踩雷**的找法：
- 先在 **Controls** 搜：`Place Backpack` / `Backpack` / `Sneak`
- 或看背包 tooltip 是否有「放置/拾取」提示

找不到也沒關係：**先把它當成穿戴式背包**用，已經能解決 80% 的整理痛點。

## 好玩的點在哪
- **探索節奏變順**：背包滿了不用立刻回家，口袋裡就能分類。
- **減少整理時間**：從「回家整理半小時」變成「邊走邊整理」。
- **死亡回收更有條理**：常用物資集中在背包，比散在全身各格好找。
