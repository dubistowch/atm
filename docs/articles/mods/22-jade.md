# Jade（資訊 HUD）新手必裝：看方塊資訊、找模組來源、除錯機器狀態

## 簡介
Jade 是方塊/實體資訊顯示模組（WAILA/HWYLA 類）：你把準星指向方塊或生物時，畫面上會即時顯示名稱、所屬模組，以及（若有整合）容器內容摘要、機器能量/流體/進度、村民職業等資訊。

在整合包裡它像「隨身檢視器」：做自動化、排管線、找方塊來源時會省非常多時間。

*圖：準星指向方塊/生物時，Jade 會在畫面上方顯示名稱、所屬模組與延伸資訊。重點是「用模組來源快速定位配方/指南」與「用能量/流體/進度資訊除錯」。*

## 模組特色
- 顯示方塊名稱、所屬模組（整合包東西多時超重要）。
- 可顯示容器/機器狀態（取決於該模組是否支援 Jade 整合）。
- 多數顯示項目可開關、可調位置與縮放。

## 按鍵操作
Jade 通常會提供「開啟設定/顯示設定」的鍵位，但整合包可能重綁；**請以遊戲內控制設定為準**。

- 打開「選項 → 控制設定」，搜尋：
  - `Jade`
  - `Waila`

建議把 Jade 的設定鍵綁到你常用但不衝突的按鍵，方便臨時調整。

## 遊戲內設定與調整
1) **先把「顯示模組名稱」打開**
- 整合包裡最常見的困擾是：看不出來這台機器是哪個模組的。
- Jade 顯示來源後，你查 JEI/REI、翻指南書都會快很多。

2) **容器內容摘要：整理倉庫與排物流很省事**
- 如果整合包啟用了容器摘要：你不用一個個打開箱子，就能看出裡面是什麼。

3) **機器能量/流體/進度：除錯神器**
- 管線接錯、供電不足、輸出滿了，很多時候 Jade 一眼就能看出原因。
- 若某模組機器不顯示：通常是「該模組沒有提供整合」或需要額外插件（依整合包）。

4) **畫面太擋：調整位置/縮放/透明度**
- 到 Jade 設定把：
  - Scale（縮放）
  - Anchor/Position（位置）
  - Opacity（透明度）
  調到不干擾戰鬥與建築的程度。

## 如何獲得模組道具
Jade 通常是純 HUD/UI 模組，不以新增可合成道具為主。

你需要做的只有：
- 設定好控制鍵
- 在 Jade 設定裡調出你想看的資訊

## 好玩的點在哪
- **學模組更快**：看到方塊立刻知道來源與狀態，少走冤枉路。
- **自動化除錯效率大增**：供電/輸入/輸出/滿格問題，很多都不用開 GUI。
- **多人教學更方便**：跟新手說「看 Jade 顯示的那行」會比口述快很多。

## 安裝與前置（可略讀）
- Jade 常有額外插件/整合模組（用來顯示更多機器資訊）；整合包通常會一併處理。
- 若你是管理員：可用設定控制顯示項目與權限（例如伺服器是否允許顯示容器內容），依版本而定。
