# Jade（方塊資訊顯示）指南

> 目標：在畫面上即時顯示你正在看/指著的方塊與實體資訊（類似 WAILA/HWYLA/The One Probe）。

## Jade 能幫你做什麼

你把準星指向某個目標時，Jade 會在螢幕上顯示：

- 方塊名稱、所屬模組
- 容器/機器的基本狀態（例如能量、流體、進度、庫存概況；取決於整合）
- 生物/村民的資訊（血量、職業等；取決於整合）

在整合包中它非常像「隨身檢視器」，常用於：

- 判斷這台機器是誰家的、需要什麼能量
- 確認管線連接是否正確
- 不打開 GUI 就看見目前進度/存量

## 基本操作

- **看資訊**：對準方塊即可。
- **進階設定**：通常有一個 Jade 設定鍵（Options → Controls 搜尋 `Jade`）。
  - 可調整顯示位置、顯示內容、是否顯示模組名稱等。

> 建議：把 Jade 設定鍵綁到你常用但不衝突的按鍵，方便臨時開關資訊。

## 推薦設定（實用向）

不同整合包選項名稱可能不同，以下是常見方向：

- **顯示 Mod 名稱**：開啟。整合包東西多，知道來源很重要。
- **顯示容器內容摘要**：開啟。看箱子/抽屜裡是什麼，不用一個個打開。
- **顯示機器能量/流體**：開啟。排管線與供電時省大量時間。
- **顯示方塊完整 ID（進階）**：需要做黑名單/自動化過濾時很有用。

## 與其他模組的高效搭配

- **Easy Villagers**：
  - 看交易站/繁殖器狀態，快速確認是否有村民、是否在運作。
- **FTB Ultimine**：
  - 挖之前先看清楚方塊是什麼（尤其多種石頭/礦變體的世界）。
- **儲存/管線模組**：
  - Jade 常能顯示管線連接與輸入輸出面（視相容性）。

## 常見問題

### Q1：資訊擋住畫面、太大？

- 到 Jade 設定調整：
  - 縮放（Scale）
  - 位置（Anchor/Position）
  - 透明度（Opacity）

### Q2：某些機器不顯示能量/進度？

- 取決於該模組是否提供 Jade 整合。
- 有時需要額外的「插件」或在設定中啟用某些顯示項。
