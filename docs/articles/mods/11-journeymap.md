# JourneyMap 教學與玩法筆記

## 簡介（這個模組在整合包的角色）
JourneyMap 是最經典也最常見的地圖模組之一，提供 **小地圖、全螢幕地圖**，以及可選用的 **Web 地圖**。在大型整合包裡，它通常就是你的「探索與導航中樞」：找生態域、找建築、標記資源點、死亡點回收、多人伺服器共享座標，都靠它提升體驗。

## 模組特色（條列，含 5-10 點）
- 小地圖（Minimap）與全螢幕地圖（Full-screen map）同步記錄你探索過的區域。
- 支援 Webmap：可在瀏覽器查看地圖（需在設定中啟用）。
- 路標系統（Waypoints）：可建立、自訂顏色/圖示、管理與分享。
- 顯示玩家、村民、動物、怪物等實體標記（可設定顯示與篩選）。
- 小地圖可切換地圖類型、縮放、預設組合（presets）。
- 設定選項完整：地圖風格、資訊欄位、標記顯示、Webmap 行為等都能調。

## 按鍵操作（列常用鍵位；若無固定鍵位，說明到「控制設定」中搜尋關鍵字；不要瞎猜）
以下為官方文件列出的預設鍵位（可在控制設定中自訂）：
- `J`：顯示/隱藏全螢幕地圖
- `Ctrl + J`：顯示/隱藏小地圖
- `=` / `+` / `-`：小地圖縮放（放大/縮小）
- `[`：切換小地圖顯示的地圖類型
- `|`：在不同小地圖預設之間切換
- `B`：在目前位置建立路標（Waypoint）
- `Ctrl + B`：開啟路標管理器（可能與旁白鍵位衝突，可自行更改）
- `Z`：切換所有路標的可見性

來源：JourneyMap 文件「Basic Usage → Key Mappings」。

## 如何設定（含常見設定檔位置/遊戲內設定入口）
- 遊戲內設定入口：
  - 進入世界後打開 JourneyMap 的設定管理器（通常可由地圖介面進入），或依文件指引使用 Options Manager。
- 常見設定檔位置（依版本可能不同）：
  - `config/journeymap/`（常見會有多個設定檔）
  - 部分設定/資料也可能在 `journeymap/` 資料夾中（例如地圖快取、路標資料；依版本而異）。
- 建議先調的幾個方向：
  - 小地圖位置/大小、實體標記顯示（避免太雜）、路標顯示、Webmap 是否啟用與埠號/權限。

## 如何獲得模組道具（生存模式取得路線：合成、掉落、交易、探索；舉 3-6 個代表性物品）
JourneyMap 主要是 UI/功能模組，通常**不以新增大量可合成道具為主**。
- 你最常用到的是：
  - **地圖介面本身**（小地圖/全螢幕地圖）
  - **路標（Waypoint）**：使用快捷鍵建立與管理
  - **死亡路標（Death Waypoint）**：死亡後自動生成（可在設定中調整顯示/保留）

> 若你的整合包額外加了 JourneyMap 相關物品（少見），請以 JEI/REI 查配方為準。

## 好玩的點在哪（給玩家的玩法建議與小技巧）
- 把路標當「冒險存檔點」：村莊、地牢入口、傳送點、資源礦洞入口都先標起來。
- 多人伺服器先統一命名規則：例如 `基地-主倉庫`、`挖礦-深板岩入口`，避免路標爆炸後找不到。
- 小地圖標記別開太多：怪物/動物標記全開在後期會很亂，建議做情境化 presets（探索/戰鬥/建家）。
- Webmap 很適合「第二螢幕」：邊跑圖邊看瀏覽器地圖超方便（但注意伺服器規範與隱私）。
