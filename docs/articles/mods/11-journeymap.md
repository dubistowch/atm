# JourneyMap 新手教學與玩法筆記

## 簡介
JourneyMap 是最常見的地圖模組之一，提供小地圖、全螢幕地圖、路標（Waypoints），部分版本/整合包還能啟用 Web 地圖。

在大型整合包裡，它就是你的「探索與導航中樞」：找生態域、找結構、標基地與資源點、死亡點回收、多人生存共享座標，都會變得很順。

![JourneyMap 小地圖與全螢幕地圖示意（官方文件圖）](https://teamjm.github.io/journeymap-docs/latest/img/background.jpg)

*圖：JourneyMap 官方文件的示意圖。重點是它同時提供「小地圖」與「全螢幕地圖」，並能配合路標（Waypoints）把探索資訊整理成可重用的導航資產。*

## 模組特色
- 小地圖 + 全螢幕地圖同步記錄探索過的區域。
- 路標系統：建立/分類/顏色圖示/分享。
- 可調整顯示：玩家、動物、怪物、村民等標記（依設定）。
- 多種地圖顯示樣式、縮放與預設組合（presets）。
- （可選）Webmap：用瀏覽器看地圖（依整合包/伺服器設定）。

## 按鍵操作
JourneyMap 的鍵位常會被整合包重綁；**請以遊戲內控制設定為準**（不要照抄固定鍵）。

- 打開「選項 → 控制設定」，搜尋：
  - `JourneyMap`
  - `Map`
  - `Waypoint`
  - `Minimap`

建議你把「開地圖」「新增路標」「開路標管理」這三個動作設成不衝突且好按的鍵。

## 遊戲內設定與調整
1) **在地圖介面內調整顯示（最常用）**
- 打開全螢幕地圖後，通常能找到設定/齒輪/選單（依版本 UI）。
- 新手最推薦先調：
  - 小地圖大小與位置
  - 是否顯示生物/怪物標記（避免畫面太雜）
  - 路標顯示規則（只顯示重要的）

2) **路標命名規則：先訂就不會爆炸**
- 推薦格式：`分類-名稱`，例如：
  - `基地-主倉庫`
  - `挖礦-深板岩入口`
  - `下界-堡壘入口`
  - `資源-史萊姆農場`

3) **死亡點與回收習慣**
- JourneyMap 多數情況會自動記死亡點。
- 建議：回收完就把死亡路標刪掉或歸檔，避免地圖變成紅點海。

4) **Webmap（若整合包有啟用）**
- Webmap 很適合雙螢幕，但也可能牽涉伺服器規範與隱私。
- 若你找不到 Webmap 選項：可能是整合包沒開、或伺服器禁用。

> 補充（只留短註）：JourneyMap 的資料與快取會寫到本機資料夾；一般玩家不用手動整理。

## 如何獲得模組道具
JourneyMap 主要是 UI 功能模組，通常不以新增可合成道具為主。

你最常用到的是：
- 地圖介面本身（小地圖/全螢幕地圖）
- 路標（Waypoint）與路標管理
- 死亡路標（Death Waypoint，依設定）

## 好玩的點在哪
- **探索不再迷路**：跑遠了也能找到回家的路。
- **任務/素材效率暴增**：看到需要某地形或結構，地圖 + 路標就能形成你的探索資料庫。
- **多人遊玩更順**：共享座標、集合點、公共資源點都能管理得很清楚。

## 安裝與前置（可略讀）
- JourneyMap 是純客戶端體驗型模組，但伺服器可能會限制某些顯示/共享功能（例如玩家雷達、Webmap）。
- 若你遇到鍵位衝突或功能消失，優先以整合包公告/伺服器規則為準。
