# Waystones 新手教學與玩法筆記

## 簡介
Waystones 提供「傳送石碑」玩法：你啟動過的 Waystone 會被記錄，之後就能在石碑之間傳送，或用卷軸/傳送石等道具遠端傳送。

在整合包裡它常用來取代 `/home`、`/warp` 這類指令便利性，讓生存也能有「合理的快速移動」。

![Waystones 模組示意圖（官方網站）](https://mods.twelveiterations.com/img/minecraft-bedrock/waystones.jpg)

*圖：Waystones 的官方示意圖。重點是「啟動過的 Waystone 會進入清單」，之後就能在各點之間傳送；實務上最常用在基地、主城樞紐與資源點之間的往返。*

## 模組特色
- Waystone 方塊：啟動後可作為傳送目的地。
- 可透過物品（卷軸/傳送石等，依版本）或 Waystone 本體傳送。
- 常見支援「全球 Waystone」（公共樞紐/主城用）。
- 村莊可能生成 Waystone（依整合包世界生成設定）。

## 按鍵操作
Waystones 的互動多在 GUI 內完成；若整合包啟用了「背包介面傳送按鈕/快捷」，才可能出現鍵位。

- 打開「選項 → 控制設定」，搜尋：
  - `Waystones`
  - `Warp`

## 遊戲內設定與調整
1) **先養成「到新地方就立刻啟動」的習慣**
- 第一次到村莊、地牢入口、重要資源點：
  - 放一個 Waystone（或找現成的）
  - 啟動並重新命名
- 這會直接改變你整個探索節奏。

2) **命名規則：讓傳送清單好用**
- 建議格式：`維度-用途-地點`，例如：
  - `主世界-基地-主倉庫`
  - `主世界-資源-礦場入口`
  - `下界-刷怪-堡壘入口`

3) **傳送成本與冷卻（看整合包）**
- 有些整合包會用經驗/物品/冷卻來平衡傳送。
- 你可以在 Waystone 的介面提示、或整合包任務/說明中找到規則。

4) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Waystones → `Config/設定`（若有）
  - 找不到就先不用管；玩家端通常不需要改。

> 補充（只留短註）：跨維度、全球 Waystone 等屬性常在設定檔調整；一般玩家照整合包規則玩即可。

## 如何獲得模組道具
常見取得方式（配方以 JEI/REI 為準）：
- Waystone：多數整合包可合成；也可能在村莊生成。
- 傳送卷軸/回程卷軸：消耗型，適合緊急回家或回樞紐。
- 傳送石（Warp Stone，依版本）：可重複使用/可充能的傳送道具。

## 好玩的點在哪
- **探索回報更高**：跑到很遠也不怕「回不來」或「死了撿不到」。
- **多人集合超方便**：公共樞紐 + 全球 Waystone（若伺服器開放）能大幅降低集合成本。
- **搭配地圖模組更強**：地圖標路標、Waystone 走傳送，跑圖效率暴增。
