# Create 新手教學與玩法筆記

## 簡介
Create 是「看得見、會動的科技」：你不是把材料丟進一堆 GUI，而是用齒輪、皮帶、轉速與各種機械方塊，在世界裡搭出真正會運轉的工廠。它特別適合拿來做前中期自動化、物流整線，以及「邊實用邊好看」的基地產線。

## 模組特色
- 以**旋轉動力**驅動一切，重視傳動設計（轉速、扭力/壓力等概念依版本顯示）。
- 大量可視化機械：皮帶、齒輪箱、動力臂、運輸與分流都能直接看到運作。
- 內建 **Ponder** 教學：在遊戲內看動畫學怎麼放、怎麼接。
- 物流玩法強：搬運、篩選、分流、堆疊、整線都能做。
- 多樣加工鏈：切割、壓縮、攪拌、洗礦等（依版本/整合包）。
- 零件本身就很適合做裝飾，工廠可以直接當建築作品。

## 按鍵操作
Create 的鍵位常被整合包調整；**請以遊戲內控制設定為準**。

- 打開「選項 → 控制設定」，搜尋：
  - `Create`
  - `Ponder`
  - `Goggles`（護目鏡）
  - `Wrench`（扳手）

如果你找不到任何鍵位，不代表不能玩；Create 很多操作是「手持物品右鍵」或「在介面裡點按鈕」。

## 遊戲內設定與調整
1) **把 Ponder 當成第一優先的教學入口**
- 看到新方塊/新物品：
  - 先在物品說明、JEI/REI 或相關介面找看看是否有 `Ponder`/教學提示（依版本呈現不同）。
- Ponder 最適合解決：「這個方塊要怎麼接動力、怎麼輸入輸出、會不會卡料」。

2) **護目鏡資訊顯示（若整合包有啟用）**
- 護目鏡通常用來顯示轉速/壓力/狀態等資訊。
- 若你覺得畫面太滿或看不到資訊：
  - 先檢查控制設定是否有 Create/護目鏡相關切換。
  - 再看看是否能在 Mods → Create → 設定裡調整顯示（依整合包與版本可能不同）。

3) **機械方塊的「行為」大多在方塊本身或其介面調**
- 例如：分流條件、篩選物品、搬運方向、停止/啟動條件，通常都能在方塊上右鍵或開 GUI 設定。
- 建議你做一個「測試牆」：把常用零件各放一個，想不起來就回來看設定，不用每次都重查。

4) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Create → `Config/設定`（如果有）。
  - 沒有 GUI 就先不用管；一般玩家靠 Ponder + 方塊介面就夠用了。

> 補充（只留短註）：Create 的動力平衡、部分行為開關可能在 config；一般玩家不需要改檔案。

## 如何獲得模組道具
以生存常見的入門順序整理（配方請用 JEI/REI）：

- **先做基本工具**
  - 扳手：調方向、拆裝與設定的核心工具。
  - 護目鏡：看資訊、查瓶頸很有用。

- **建立第一條動力主幹**
  - 一個穩定動力來源（依整合包提供的選項）
  - 傳動軸、齒輪、皮帶輪等基本傳動元件

- **選一條你最需要的加工/自動化當第一個專案**
  - 例如：食物加工線、木材量產、基礎材料加工、簡單分流倉儲等。

代表性物品（建議先用 JEI 搜尋確認名稱）：
- 扳手（Wrench）
- 護目鏡（Goggles）
- 傳動軸/齒輪/皮帶相關元件
- 你整合包版本的基礎加工方塊（切割/壓縮/攪拌…）

## 好玩的點在哪
- **先做一條「漂亮的主傳動」**：把動力幹線規劃好，再分支出各條產線，整個工廠會很有秩序。
- **產線可視化的爽感**：材料從 A 端被送上皮帶、一路加工、最後進箱子，你會很清楚每一步在幹嘛。
- **追求可維修性**：留走道、留檢修口、分段設計，後期擴建超省事。
- **工廠即建築**：用玻璃做觀景窗、把皮帶當裝飾，Create 很適合做「展示型基地」。

## 安裝與前置（可略讀）
- Create 常見會搭配許多附加模組（例如額外零件/機械/裝飾），整合包實際內容以模組清單為準。
- 一般玩家：裝好整合包後直接用 Ponder 學即可。
- 若你是整合包作者/伺服器管理員：動力平衡、部分行為開關可能在 config；修改前先備份。
