# Create 新手教學與玩法筆記

## 簡介
Create 是「美學科技」代表：它不是把玩法關在一堆 GUI 裡，而是讓你用齒輪、皮帶、轉速與各種動力元件，在世界中組裝會動的機械結構，做出搬運、加工、農場與工廠。整合包裡的 Create 常用來做「看得見、會動的自動化」，同時兼顧美觀與效率。

## 模組特色
- 用旋轉動力驅動機器，重視「傳動設計」而不只是放機器吃電。
- 可以搭建各種可視化機械結構與裝置，玩法高度自由。
- 內建視覺化教學系統 Ponder，直接在遊戲內看動畫學機械。
- 強大的搬運與物流：從物品輸送到分流、篩選與整線。
- 多樣化的加工鏈：壓縮、切割、攪拌、洗礦等（實際可用內容依版本）。
- 擅長做農場與量產：樹場、作物收割、食物加工線等都能做得很漂亮。
- 很適合與其他模組互補：Create 做前中期自動化與物流，其他科技模組做能量/後期材料。
- 以建築裝飾性見長：零件本身就很好看，適合做展示型工廠。

## 按鍵操作
Create 的鍵位在不同版本與整合包中常被調整，尤其是 Ponder 與各種工具。

- 進入「選項 → 控制設定」，搜尋：`Create`、`Ponder`、`Goggles`、`Wrench` 等關鍵字。
- 若你找不到固定鍵位，通常代表整合包沒有預設綁定，或是已被其他模組覆蓋；請在控制設定自行設定常用按鍵。

## 如何設定
Create 的設定入口與檔案位置通常包含：

- **全域設定檔**：`.minecraft/config/` 內的 Create 相關設定（常見為 `.toml`）。
- **伺服器/世界設定**：部分行為可能需要伺服器端配置才會生效（依版本可能不同）。
- **遊戲內可調內容**：例如護目鏡資訊顯示、裝置行為等，有些會在物品/方塊介面中調整。

若你是整合包作者，Create 有不少可調參數（例如動力平衡、部份配方或機制開關），建議在測試世界逐項確認。

## 如何獲得模組道具
以下以生存常見路線整理；配方請用 JEI/REI 查你整合包版本。

- **合成取得**
  - 扳手（用於調整零件方向/行為）
  - 護目鏡（用於查看資訊）
  - 基礎動力元件：手搖/動力源、齒輪、傳動軸、皮帶輪等

- **加工與量產**
  - 許多 Create 物品會鼓勵你建立加工線（例如攪拌、壓縮、洗礦等），用自動化量產零件。

代表性物品（3–6 個）：
- 扳手
- 護目鏡
- 動力元件（齒輪、傳動軸、皮帶系統）
- 基礎加工機（依版本可能是壓縮/切割/攪拌等）
- Ponder 相關教學介面（用於學習）

## 好玩的點在哪
- **先做「一條漂亮的傳動主幹」**：把動力主線規劃好，再把每條產線接出去，整個工廠會非常有秩序。
- **用 Ponder 當你的第一本教科書**：看到任何新方塊先 Ponder 一下，比查影片還快。
- **追求「可維修」而不是只追求快**：留走道、留檢修口、把分段做成模組化，後期擴建很省事。
- **把工廠當成建築作品**：Create 的魅力之一就是「產線即裝飾」。試著把管線藏進牆體、做觀景窗與展示區。
