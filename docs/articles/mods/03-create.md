# Create 新手教學與玩法筆記

## 先看這裡（前 30 分鐘）
**目標：做出第一條可擴充的旋轉動力主幹，並用 Ponder 學會 3 個加工元件。**

1) 先做：
- **Wrench（扳手）**：調方向、拆裝、設定核心工具
- **Goggles（護目鏡）**（若整合包有）：看轉速/壓力（Stress）與瓶頸

2) 建第一段動力：
- 任一動力來源（依整合包版本：水車/風車/蒸汽/引擎等）
- **Shaft（傳動軸）**、**Cogwheel（齒輪）**、**Gearbox（齒輪箱）**、**Belt（皮帶）**

3) 看到任何新方塊：先按 **Ponder** 看動畫教學，再放進世界測。

> Create 的「最快學習法」就是：**Ponder → 放一個測試線 → 看它怎麼卡**。

---

## 這模組到底強在哪
- **可視化自動化**：材料在皮帶上跑、在機器裡旋轉，問題一眼看得出來。
- **旋轉動力（Rotation）**是共同語言：你做的是「傳動設計」，不是 GUI 堆疊。
- **物流/分流很完整**：輸送、篩選、打包、堆疊、分岔都能做。
- **Ponder 內建教學**：不用背配方，也不用背機械規則，直接看動畫。

---

## 核心觀念（把這 4 件事弄懂，九成工廠就不會爛尾）
### 1) 轉速（RPM）與壓力/負載（Stress）
- 轉得動 ≠ 轉得久：負載超過就會停。
- 新手做法：
  - 先做「**主幹線**」供動力
  - 每條加工線「**分段**」接（方便查是哪段超載）
  - 用護目鏡看是哪台機器吃掉負載

### 2) 方向與傳動（90% 問題都在「方向不對」）
- 齒輪箱/齒輪/皮帶會改變方向與轉速。
- 你不確定就：**拿起扳手 → 看方塊的輸入輸出面 → Ponder**。

### 3) 產線要「可維修」
- 留走道、留檢修口、把皮帶/軸走天花板。
- 讓你可以不拆整座工廠就換一顆齒輪。

### 4) 先做「你最缺的那條線」
- Create 可以做很多花活，但新手最容易被分心。
- 先把一條產線做到穩，再擴建第二條。

## 進度清單（推薦的新手路線）
- [ ] Wrench（扳手）
- [ ]（可選）Goggles（護目鏡）
- [ ] 穩定動力來源 + 一段可擴充主幹（軸/齒輪箱/皮帶）
- [ ] 做出第一個「加工三件套」中的任意兩個：
  - [ ] **Mechanical Press（機械壓機）**（壓縮/板材常用）
  - [ ] **Mechanical Saw（機械鋸）**（木材加工、部分配方）
  - [ ] **Mechanical Mixer + Basin（攪拌機 + 盆）**（混合/洗礦/食物/材料）
- [ ] 做出一條最小物流：漏斗/輸送/簡單分流（依整合包提供元件）
- [ ] 學會用 Ponder 查：
  - [ ] 這方塊要吃哪種輸入？
  - [ ] 速度需求大概多少？
  - [ ] 它會產生什麼副產物/需要輸出空間嗎？

## 常見問題（除錯順序照抄即可）
### Q1：機器不轉 / 轉一下就停
- 用護目鏡看 **Stress 是否超載**
- 先拆掉一半負載測：確認是「供能不足」還是「方向/連接錯」
- 動力來源太小就加大（或分段供能）

### Q2：東西卡在漏斗/輸送帶上不進機器
- 確認機器的「**輸入面**」在哪（很多機器有特定面）
- 盆（Basin）類多半需要正確對齊（Ponder 有演示）
- 確認輸出端有空間：有些加工會把成品留在盆/方塊內，需要你接漏斗拉走

### Q3：轉速太慢/產線吞吐不夠
- 不要盲目加速：先確保 Stress 有餘裕
- 把加工拆成多台並聯（比單台暴力拉 RPM 更穩）

### Q4：我不知道下一步該做什麼
- 先選一個你最常做的事情，做成自動化：
  - 木材（鋸）
  - 板材/壓縮（壓機）
  - 基礎混合材料（攪拌 + 盆）
- 看到新方塊一律先 Ponder，再決定要不要投入。
