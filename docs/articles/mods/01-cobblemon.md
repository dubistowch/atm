# Cobblemon 新手教學與玩法筆記

![模組圖示](https://cdn.modrinth.com/data/MdwFAVRL/abfca1654a2d09bab85cbffcc9869938c951ee0e_96.webp)

## 簡介
Cobblemon 是把「寶可夢的捕捉、培育、對戰」直接融進 Minecraft 生存流程的模組：你一樣要採集、探索、下礦、蓋家，但世界同時會生成野生寶可夢，你可以把牠們當成隊伍成員、移動工具與長期收集目標。

在整合包裡它通常是很耐玩的「長線主線」：想補圖鑑、抓稀有、做出更好的球與培育資源，就會自然被推著去跑生態域、找結構、做基地機能。

## 模組特色
- 捕捉、培育並組成隊伍，與野生寶可夢對戰取得經驗與招式成長。
- 對戰系統採用 Pokémon Showdown 引擎，偏向還原競技規則。
- 部分版本支援「自由移動戰鬥」：戰鬥期間仍可走動（整合包/伺服器規則可能有限制）。
- 具備圖鑑（Pokédex）與掃描/資訊查閱能力，方便查詢生成與個體。
- 可放出寶可夢在世界中巡遊、展示（依版本/附加內容）。
- 莓果系統：野外生成、可種植、可堆肥加成，部分玩法可培育出新莓果。
- 化石與復活機制：探索/考古取得化石，再透過機器復活古代寶可夢（依版本）。
- 多人支援對戰、交易與交換（依伺服器規則）。

## 按鍵操作
Cobblemon 的常用操作會因版本與整合包而不同；**請以遊戲內控制設定顯示為準**，不要照抄網路文章的固定鍵位。

- 打開「**選項 Options → 控制設定 Controls**」，在搜尋框輸入以下關鍵字找出相關操作：
  - `Cobblemon`
  - `Pokedex` / `Pokédex`
  - `Pokemon` / `Pokémon`
  - `Party`（隊伍）

小提醒：若你有裝很多模組，控制設定右上角通常有搜尋欄，先搜關鍵字會快很多。

## 遊戲內設定與調整
以「玩家在遊戲裡找得到的入口」為主，給你最常用的調整方式：

1) **先找鍵位與提示**
- 路徑：選項 → 控制設定
- 作法：用上面提供的關鍵字搜尋，確認是否有「圖鑑開啟、隊伍介面、放出/收回」等操作。

2) **看得到就先用 Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - 有些版本可在「**Mods**」清單中點 Cobblemon → `Config/設定` 進入圖形化設定。
  - 若找不到按鈕，代表這版可能沒有內建 GUI，或整合包關掉了。

3) **用遊戲內教學/提示建立你的日常流程**
- 建議把基地附近先做出：補給（食物/床）、收納（箱子/背包模組）、以及一小塊「莓果田/培育區」。
- 你會發現 Cobblemon 的舒服程度，很大一部分來自「出門前準備」與「回家後整理」是否順。

4) **（可選）伺服器常見指令與規則**
- Cobblemon/伺服器可能提供治療、傳送、對戰規則等指令或插件功能，但差異很大。
- 建議直接看伺服器公告或輸入 `/help` 搜尋 Cobblemon 相關指令；**不建議在不知道規則時亂用指令**。

> 補充（只留短註）：若你是整合包作者或私服管理員，部分平衡/生成細節可能會放在 config 或資料包中調整；一般玩家通常不需要碰到檔案。

## 如何獲得模組道具
以下以「你在生存中最常走的取得路線」整理；配方與名稱請以 JEI/REI 為準。

- **合成取得（最常用）**
  - 精靈球相關道具（捕捉用）
  - 寶可夢圖鑑（Pokédex）
  - 牧場/展示相關方塊（讓寶可夢在基地中活動）

- **探索與採集（順路就做）**
  - 莓果：野外採集後帶回家種植，慢慢擴田。
  - 結構寶箱：部分整合包會把球、材料或相關物品放進戰利品池。

- **挖掘/特殊點位（中期目標）**
  - 化石：常與可疑沙/可疑礫等考古資源點或特定地形/結構相關（依版本）。
  - 復活用機器與材料：多半需合成，並搭配探索取得的化石。

## 好玩的點在哪
- **探索變得「有方向」**：不是亂跑，而是先決定要抓的屬性/招式/變體，再反推去哪個生態域、哪個時間點更適合。
- **基地規劃出「訓練—補給—收納」動線**：回家先治療/整理→補球與食物→再出門，整體節奏會很舒服。
- **多人社交自然發生**：交易、對戰、交換進化，會讓伺服器社群更活。
- **收集控的長線目標**：圖鑑、稀有外觀、古代寶可夢、特定個體（性格/個體值）都很耐刷。

## 安裝與前置（可略讀）
- Cobblemon 可能有不同載入器與版本分支（依整合包而定），請以你整合包的模組清單為準。
- 一般玩家只需要確認：整合包已正確安裝、伺服器與用戶端版本一致、資源包/材質包（若有要求）已啟用。
- 若你需要做管理員級調整：可能會用到 config / 資料包（datapack）來改生成、戰利品或平衡；動手前先備份存檔與設定。
