# Cobblemon 新手教學與玩法筆記

## 簡介
Cobblemon 是一款把寶可夢玩法「融進」Minecraft 生存流程的模組：你仍然需要採集、探索、建家與冒險，但同時可以捕捉、培育與對戰寶可夢，並把牠們當作移動、探索與多人互動的一部分。在整合包中，它通常扮演「長線目標」與「探索驅動」的核心內容：想要圖鑑、稀有掉落與特殊結構，就得出門跑圖與規劃隊伍。

## 模組特色
- 可捕捉、培育並組成隊伍，與野生寶可夢對戰取得經驗與招式成長（已實作大量世代寶可夢）。
- 對戰機制採用 Pokémon Showdown 引擎，偏向還原競技規則。
- 戰鬥時可選擇「自由移動」模式：你可以在戰鬥期間走動、挖掘、甚至處理周遭怪物。
- 具備寶可夢圖鑑與掃描功能，方便查閱與觀察野生個體。
- 可騎乘寶可夢進行移動與探索（依版本更新內容可能有所不同）。
- 多人遊戲支援對戰與交易。
- 牧場等機制讓寶可夢在世界中放出、巡遊與展示。
- 莓果系統：野外生成、可種植、可堆肥加成，並可交配培育出新莓果。
- 化石與復活機制：挖掘可疑沙礫取得化石，再透過機器復活古代寶可夢。
- 世界生成包含古代結構與寶箱戰利品，鼓勵長距離探索。

## 按鍵操作
Cobblemon 的常用操作會因版本、是否安裝附加模組而不同；建議以遊戲內控制設定為準。

- 進入「選項 → 控制設定」後，在搜尋框輸入：`Cobblemon`、`Pokedex`、`Pokédex`、`Pokémon` 等關鍵字，查看目前整合包實際綁定的鍵位。
- 若你使用的是控制器或改鍵整合包，也建議在同一處確認是否被覆蓋。

> 注意：不要依網路文章盲目照抄鍵位，很多整合包會重綁。

## 如何設定
Cobblemon 常見的調整方式分為三類，實際檔名與路徑可能依載入器與版本不同。

1. **模組設定檔**
   - 伺服器/單機通常在 `.minecraft/config/` 底下出現對應的設定檔（可能是 `.toml`、`.json`）。
   - 部分設定可能放在專屬資料夾（例如 `config/cobblemon/` 之類）。

2. **資料包與資源包**
   - Cobblemon 支援用資料包、資源包做自訂內容（例如自訂變體、fakemon 等）。
   - 生存世界內可用資料包資料夾：`你的存檔/datapacks/`。

3. **整合包層級調整**
   - 若整合包作者有放入預設設定，可能會在 `defaultconfigs/`（Forge/NeoForge 常見）或自訂覆寫機制中。

若你要做較大的客製（新增/替換自訂寶可夢資料），建議先閱讀官方教學頁面，並在測試世界驗證後再放進主存檔。

## 如何獲得模組道具
以下以「取得途徑」來整理，實際配方請以 JEI/REI 查詢你所在整合包版本。

- **合成取得**
  - 精靈球相關道具（用於捕捉）
  - 寶可夢圖鑑（Pokédex）
  - 牧場/展示相關方塊（讓寶可夢在基地中活動）

- **探索與採集**
  - 莓果：可在野外找到，也可移植回家種植、交配培育。
  - 古代結構：探索結構中的寶箱可能出現模組相關戰利品。

- **挖掘與特殊掉落**
  - 化石：通常與可疑沙/礫等考古型資源點相關（依版本/整合包設計）。
  - 復活流程所需的材料與機器：多半需合成並搭配探索獲得的化石。

代表性物品（建議先用 JEI 搜尋確認名稱與配方）：
- 精靈球系列
- 寶可夢圖鑑
- 莓果與堆肥/培育相關物品
- 牧場相關方塊
- 化石與復活機器

## 好玩的點在哪
- **把探索變成「抓怪路線」**：先決定想抓的屬性/招式，再反推要去哪些生態域與結構，探索更有目標。
- **基地規劃一個「培育農場」**：莓果田、治療與補給、牧場展示區，一次把生活品質拉滿。
- **多人伺服器的社交玩法**：交易、對戰、交換進化，能自然形成玩家社群。
- **自由移動戰鬥的戰術**：如果整合包允許，你可以在戰鬥時利用地形、順手處理周遭資源或危險怪物（注意安全與規則）。
- **長線目標**：以完成圖鑑、收集稀有變體、打造專屬基地展示為目標，很耐玩。
