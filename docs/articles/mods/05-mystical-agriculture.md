# Mystical Agriculture 新手教學與玩法筆記

![模組圖示](https://cdn.modrinth.com/data/C95ReXie/2bddd50d346a470536319c9c1ae6c21f6a5b3e72_96.webp)

## 簡介
Mystical Agriculture 的核心概念很直白：**用作物來種出資源**。你不只是在種小麥，而是把礦物、材料、怪物掉落做成「種子」，透過農田量產精華（Essence），再合成你需要的資源。

在整合包裡，它通常是中後期的「材料壓力終結者」：當你需要大量鐵、金、紅石、模組材料，或某些麻煩掉落時，種田會比一直跑圖或刷怪更穩定。

## 模組特色
- 大量資源作物：涵蓋原版與常見模組材料（依版本/整合包）。
- 以 Essence 合成材料：產能可擴充、好控制。
- 成長加速系統：成長加速器、澆水等機制（依版本/整合包）。
- 便利收割與自動化：鐮刀、收割機等做法（名稱依版本）。
- 種子製作有明確進度：從低階精華一路升階，逐步解鎖高階資源。
- 多數整合包會提供遊戲內指南（常見需要 Patchouli）。

## 按鍵操作
Mystical Agriculture 通常沒有「必定存在」的核心鍵位，多數互動是右鍵/GUI。

- 打開「選項 → 控制設定」，搜尋：
  - `Mystical`
  - `Agriculture`
  - `Cucumber`（常見前置）

若沒有找到任何鍵位也正常：表示主要操作都在方塊介面與合成流程。

## 遊戲內設定與調整
1) **先用 JEI/REI 把「你這包」的種子流程看一遍**
- 每個整合包最常改的就是：種子成本、祭壇需求、精華升階難度。
- 你可以先挑 3 種最常缺的資源（例如鐵/金/紅石）當第一批目標。

2) **把農場做成「可擴充模組化」**
- 建議規劃：
  - 一區一種作物（清楚好維護）
  - 產物先進緩衝箱/大容量儲存（避免爆倉）
  - 再接進倉庫系統/合成系統

3) **成長加速與收割方式的選擇**
- 若整合包有提供成長加速器/加速方塊：先做一小套測試，看看耗能與產量是否划算。
- 收割請先從「不會打掉作物」或「可控」的方法開始，避免田一大就變成維修地獄。

4) **Mod 設定畫面（若有）**
- 依整合包與版本可能不同：
  - `ESC` → **Mods** → Mystical Agriculture → `Config/設定`（若有）
  - 若沒有 GUI，玩家通常也不需要改。

> 補充（只留短註）：種子/配方/平衡很多時候是整合包用資料包或腳本改的，玩家只要照 JEI/REI 與任務線走即可。

## 如何獲得模組道具
常見入門路線（配方以 JEI/REI 為準）：

- **先下礦找起始礦物**
  - 多數版本會需要 Prosperity、Inferium 等起始資源（名稱依版本）。

- **建立種子製作核心**
  - Seed Base（多數種子都需要）
  - 合成/儀式方塊（如 Infusion Altar + Pedestals 類；名稱依版本）

- **開始量產低階精華，再慢慢升階**
  - 一旦低階精華供應穩定，後面所有種子都會順很多。

代表性物品（建議用 JEI 搜尋）：
- Prosperity/Inferium 相關材料
- Seed Base
- Infusion 類祭壇與基座（依版本）
- 成長加速器/加速方塊（若有）
- 收割/播種用機器（若整合包搭配其他農場模組也常用）

## 好玩的點在哪
- **把「卡材料」變成「擴農場」**：缺什麼就種什麼，目標非常清楚。
- **中後期的安心感**：不用再為了某個材料跑 3 個地形、刷 2 小時。
- **農場越做越漂亮**：把每一區作物做成溫室/田園/工業農場，既實用又有成就感。

## 安裝與前置（可略讀）
- Mystical Agriculture 在很多整合包會搭配 Patchouli 類指南系統、以及其他農場/儲存模組一起使用；實際內容以整合包為準。
