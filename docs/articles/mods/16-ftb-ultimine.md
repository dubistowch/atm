# FTB Ultimine（連鎖挖掘）指南

> 目標：用一個按鍵，把「挖一格」升級成「一次挖一片」；大幅縮短採礦、整地、砍樹、清理礦脈的時間。

## 這個模組解決什麼問題？

- 挖礦時遇到大型礦脈（煤、鐵、銅、紅石、石英等）要一格一格挖很慢。
- 整地/挖水道/清理地形很耗時間。
- 砍樹、挖沙礫、清理下界地獄岩時，耐久消耗與時間成本太高。

FTB Ultimine 提供：

- **按住熱鍵**，在挖方塊的同時自動連鎖挖掘相鄰方塊。
- 可切換不同模式（礦脈/方塊類型/範圍形狀等），並限制最大連鎖數量。

## 基本操作

1. 打開按鍵設定（Options → Controls），搜尋 **Ultimine**。
2. 設定/記住：
   - **啟動鍵（通常是按住）**
   - **模式切換鍵**（Cycle Mode）
   - （若有）**設定介面**（Open Config / Settings）
3. 遊戲中：
   - 手持工具（鎬/斧/鏟）
   - **按住啟動鍵**並挖一格，會自動挖掉符合條件的相鄰方塊。

> 小提醒：不同整合包會把熱鍵改成不同按鍵；請以你自己的 Controls 為準。

## 常見模式與適用情境

### 1) 礦脈模式（Ore / Vein）

- 用途：挖礦最常用。
- 行為：只連鎖挖掘同一種方塊（例如整條煤礦脈）。
- 推薦：前中期採礦、下界挖石英。

### 2) 相同方塊模式（Same Block）

- 用途：清理大量同材質方塊，例如：
  - 砂岩、石頭、安山岩
  - 地獄岩、末地石
- 風險：很容易「挖過頭」造成洞太大；建議搭配較低的最大連鎖數。

### 3) 形狀/範圍模式（Shapeless / Tunnel / Plane… 視整合而定）

- 用途：整地、開礦道、挖 3x3 或平面。
- 建議：確保你有足夠照明與背包空間，避免一次把支撐挖掉造成塌方（砂礫/沙子）。

## 重要設定建議（強烈推薦調整）

以下通常可以在遊戲內 Ultimine 設定或模組設定檔找到：

- **Max Blocks（最大連鎖數）**：
  - 前期：32–64
  - 中期：64–128
  - 後期：128–256（看你的工具與包內平衡）
- **Require Tool（需要正確工具）**：建議開啟，避免徒手把整片挖掉。
- **Affects Durability（消耗耐久）**：通常會照挖掘數量消耗；請準備修補或更換工具。
- **Use Hunger（消耗飢餓）**：若整合包開啟，連鎖會吃飽食度；帶食物。
- **Blacklist / Whitelist（黑白名單）**：
  - 把「不想誤挖」的方塊加入黑名單（例如基礎機器、基地建材）。

## 實戰技巧

- **採礦**：礦脈模式 + 64 上限，效率最高又不容易挖穿基地。
- **砍樹**：用斧頭連鎖木頭；注意葉子不一定會一起清。
- **挖沙礫**：小心重力方塊；建議從上往下挖，避免頭頂塌落。
- **下界挖石英**：效率極高，但請小心岩漿與洞口。

## 與其他模組的互動

- **Jade**：可快速辨識正在挖的方塊種類，避免挖錯。
- **背包/儲存模組**：連鎖挖會產生大量掉落物，準備好抽屜/背包/磁鐵類物品。
- **自動整理（如 Sophisticated Backpacks）**：搭配過濾器，把礦石、石頭、碎片自動分類。

## 常見問題

### Q1：為什麼沒有連鎖？

- 檢查是否：
  - 沒按住啟動鍵
  - 模式不對（例如在礦脈模式挖石頭）
  - 工具不符合（需要鎬但你拿斧）
  - 最大連鎖數太低

### Q2：會不會太 OP？

- 整合包通常用以下方式平衡：
  - 消耗耐久/飢餓
  - 限制最大連鎖數
  - 需要特定工具等級

你也可以透過設定把它調到「舒服但不破壞平衡」。
