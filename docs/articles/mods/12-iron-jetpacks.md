# Iron Jetpacks 教學與玩法筆記

## 簡介（這個模組在整合包的角色）
Iron Jetpacks 提供以 FE（Forge Energy）驅動的噴射背包，讓你在中前期就能享受「垂直移動」的自由：上山、挖礦直上直下、建築拉線、跨地形探索都變得非常順手。多數整合包會把它當作早期到中期的移動解法，後期則可能被更高階的飛行裝備取代，但依然是實用、直覺、容易上手的選擇。

## 模組特色（條列，含 5-10 點）
- 以 FE 供能的噴射背包，通常有多個等級（材料越高階越強）。
- 強調「可自訂」：速度、耗能、推力等可透過設定或資料檔調整（依版本/整合包）。
- 具備文件與 wiki，方便整合包作者調平衡、玩家查資料。
- 與大多數能源供應模組相容（只要能提供 FE）。
- 適合建築、探索、挖礦動線優化，能顯著減少跑酷與爬梯時間。

## 按鍵操作
噴射背包類模組常見會有「啟動/懸停/模式切換」等按鍵，但 **不同版本與整合包可能修改預設鍵位**。

- 請到：**選項 → 控制**
- 搜尋：`Iron Jetpacks` 或 `Jetpack`
- 將常用功能（例如啟動、懸停/穩定模式、引擎開關等）改成你順手且不衝突的鍵位。

> 本篇不直接列出預設鍵位，避免因版本差異造成誤導。

## 如何設定（含常見設定檔位置/遊戲內設定入口）
- 常見設定檔位置（依版本可能不同）：
  - `config/ironjetpacks/` 或 `config/ironjetpacks-*.toml`
  - 若整合包使用資料驅動，可能會有 Jetpack JSON 之類的定義檔（見官方文件）。
- 官方文件站（偏整合包/開發者）：
  - 文件首頁提到可用 Jetpack JSON 與配方用途設定。

## 如何獲得模組道具（生存模式取得路線：合成、掉落、交易、探索；舉 3-6 個代表性物品）
一般以合成取得為主：
- **基礎噴射背包（低階）**：用較便宜的金屬與簡單元件合成，作為入門飛行。
- **中高階噴射背包**：逐階升級，通常需要更好的金屬、能量元件與更多 FE。
- **噴射背包升級/零件**：有些版本會提供強化用元件或升級路線（依整合包配置）。
- **充電方式**：通常要靠你的能源系統（發電機、電池、充電器/充能方塊等）補能。

> 具體物品名稱與等級（例如鐵、金、鑽石等）會因版本與整合包配方而不同，請以 JEI/REI 顯示為準。

## 好玩的點在哪（給玩家的玩法建議與小技巧）
- 先把「供電與充電」搞定：噴射背包最怕飛到一半沒電；準備一個穩定的充電點或攜帶式電池。
- 建築神器：拉管線、架高塔、蓋大型平面時效率大幅提升。
- 挖礦動線：直上直下穿梭坑道很快，但注意別在岩漿上方沒電。
- 跟地圖/路標模組搭配：飛行 + 路標探索，跑圖節奏會非常舒服。
